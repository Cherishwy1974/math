偶函数（Y轴对称）：
$\cos(-x) = \cos(x)$ ✓
$\sec(-x) = \sec(x)$ ✓

奇函数（原点对称）：
$\sin(-x) = -\sin(x)$ ✓
$\tan(-x) = -\tan(x)$ ✓
$\csc(-x) = -\csc(x)$ ✓

倒数关系：分母的秘密

$$\sec(x) = \frac{1}{\cos(x)}$$
当$\cos(x) = 0.5$时：$\sec(x) = \frac{1}{0.5} = 2$

当$\cos(x) = 0$时：$\sec(x) = \frac{1}{0} = $未定义！

这解释了为什么$\sec$的定义域要排除$\cos=0$的点

值域限制：倒数的影响

从$[-1,1]$到$(-\infty,-1] \cup [1,+\infty)$
cos范围
→
sec范围
$\cos(x)$的值域：$[-1,1]$
当$|\cos(x)| = 1$时：$|\sec(x)| = 1$
当$|\cos(x)| = 0.5$时：$|\sec(x)| = 2$
当$|\cos(x)| = 0.1$时：$|\sec(x)| = 10$
当$|\cos(x)| \to 0$时：$|\sec(x)| \to \infty$

所以$\sec(x)$永远$\geq 1$或$\leq -1$

主值区间：反三角的妥协

为了唯一性的牺牲
$\sin(\pi/6) = 0.5$
$\sin(5\pi/6) = 0.5$
$\sin(13\pi/6) = 0.5$
...
无数个角度都得到0.5！

所以$\arcsin(0.5)$只能选一个：
$\arcsin(0.5) = \pi/6$
（限制在$[-\pi/2, \pi/2]$内）
