无穷小的阶

设 α、β 是无穷小，比值 $\lim\limits \frac{\beta}{\alpha}$ 决定阶的关系
高阶：比值 = 0
低阶：比值 = ∞
同阶：比值 = k≠0
等价：比值 = 1

理解要点

通俗解释：
• 高阶 = β 趋向 0 比 α 更快
• 低阶 = β 趋向 0 比 α 更慢
• 同阶 = 两者速度相当
• 等价 = 两者几乎一样

典型例子

α β 关系
x x² β是α的高阶
x² x β是α的低阶
x 2x 同阶
x sin x 等价

等价无穷小定理

若 α ~ α' 且 β ~ β'，则 $\lim\limits \frac{\alpha}{\beta} = \lim\limits \frac{\alpha'}{\beta'}$
意义：
计算极限时可以用等价的更简单的式子替换

常用等价无穷小

当 x → 0 时
• sin x ~ x
• tan x ~ x
• arcsin x ~ x
• arctan x ~ x
• 1 - cos x ~ x²/2
• e^x - 1 ~ x
• ln(1 + x) ~ x

替换练习

$\lim\limits_{x \to 0} \frac{\sin 2x}{\tan 3x}$
