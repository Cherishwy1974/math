区间分类实验室

区间计算器

$[3, 7]$ 包含3和7

核心理解

只看"第一个人"和"最后一个人"算不算在队伍里
[ ] = 算你一个（包含）
( ) = 不算你（不包含）

数轴可视化

5.0
5.0 在区间 [3, 7] 内 ✓

四种基本区间

[a, b]
两端都算
(a, b)
两端都不算
(a, b]
左不算右算
[a, b)
左算右不算

🚶 排队场景模拟

老师说："从张三到李四，你们都出列！"
张三要出来 ✓，李四要出来 ✓
他们之间的人也都要出来 ✓

📐 理解规则

排队 = 区间
队伍边界 = 端点
闭区间 [a,b]：
"从张三到李四都出列"
→ 张三算、李四也算

开区间 (a,b)：
"张三和李四之间的人出列"
→ 张三不算、李四也不算

半开半闭：
看括号类型决定算不算

🎯 直观记忆

方括号 [ ]
像围墙，把人围进来
"算你一个！"
圆括号 ( )
像气泡，人在外面
"不算你！"

有界区间

队伍有明确的开头和结尾
[3, 7]
3, 4, 5, 6, 7
(3, 7)
3.1, 4, 5, 6, 6.9
(3, 7]
3.1, 4, 5, 6, 7
[3, 7)
3, 4, 5, 6, 6.9
选择一个区间查看例子

特殊区间

单点集： $[a, a] = \{a\}$
只有一个人的队伍
空集： $(a, a) = \emptyset$
没有人的队伍（不可能）
全体实数： $(-\infty, +\infty)$
所有人都在队伍里

无界区间

队伍一头没有尽头
[3, +∞)
从3开始往后所有
(3, +∞)
比3大的所有
(-∞, 7]
7及以下所有
(-∞, 7)
比7小的所有
生活例子：
"[1.2, +∞)" = 身高1米2及以上才能玩
"(18, +∞)" = 必须年满18岁
"(-∞, 100]" = 100分及以下都算合格

区间运算

交集 ∩： 两个队伍都包含的人
并集 ∪： 至少在一个队伍里的人
补集： 不在队伍里的人
