复合函数求导 - 链式法则

🤔 为什么是相乘？

链式法则：导数要相乘，不是相加！
🔍 放大镜的例子：
第一个放大镜：放大 2倍
第二个放大镜：放大 3倍
两个叠加使用：放大 2 × 3 = 6倍

是相乘，不是相加！
❌ 错误理解：
不是像速度相加（60 + 2 = 62）
✅ 正确理解：
是像放大倍数相乘（2 × 3 = 6）
复合函数：$y = (2x + 1)^3$
内层变化率：2
外层变化率：$3u^2$
总变化率：$2 \times 3u^2 = 6(2x+1)^2$

⚙️ 齿轮传动原理

齿轮比喻：
• 第一个齿轮：转1圈 → 第二个齿轮转2圈
• 第二个齿轮：转1圈 → 第三个齿轮转3圈
• 总效果：第一个转1圈 → 第三个转 2×3=6圈

这就是为什么要相乘！
数学语言：
$\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx}$
（y对x的变化率 = y对u的变化率 × u对x的变化率）

📊 看斜率相乘

导数 = 切线斜率
复合函数导数 = 两个斜率相乘
1 内层函数 $g(x) = 2x$
斜率（导数）= 2
2 外层函数 $f(u) = u^2$
在 $u = 4$ 时，斜率 = 8
3 复合函数 $f(g(x)) = (2x)^2$
总斜率 = 2 × 8 = 16

💡 直观理解

想象爬山：
• 山坡倾斜度 = 30度（外层）
• 你走的路是斜的 = 2倍陡（内层）
• 实际爬升 = 坡度 × 路径陡度

两个因素相乘决定最终的陡度！

✏️ 最简单的练习

题目：求 $(3x)^2$ 的导数
1 内层是什么？
2 $3x$ 的导数是？
3 外层 $u^2$ 的导数是 $2u = 2(3x) = 6x$
最终答案：外导 × 内导
$6x \times 3 = ?$
🎉 太棒了！
记住：是 6x × 3 = 18x（相乘）
不是 6x + 3 = 6x + 3（相加）

📊 再练一个

试试：$\sin(2x)$ 的导数
提示：
• 内层：$2x$，导数 = 2
• 外层：$\sin u$，导数 = $\cos u$
• 答案：$\cos(2x) \times 2 = ?$
记住口诀：
"外导乘内导"
1. 外层函数求导
2. 乘以（不是加！）
3. 内层函数的导数
常见错误：
❌ $\sin(2x)$ 的导数 = $\cos(2x)$（忘了乘2）
✅ $\sin(2x)$ 的导数 = $2\cos(2x)$（记得乘内导）

📝 最重要的三句话

1. 导数要相乘，不是相加
2. 外导 × 内导
3. 永远不要忘记内层导数
四个最常见的：
• $(ax)^n$ → $n(ax)^{n-1} \times a$
• $\sin(ax)$ → $\cos(ax) \times a$
• $e^{ax}$ → $e^{ax} \times a$
• $\ln(ax)$ → $\frac{1}{ax} \times a = \frac{1}{x}$

✅ 最后检验

自测：$(5x)^3$ 的导数是？
思考过程：
1. 内层：$5x$，导数 = 5
2. 外层：$u^3$，导数 = $3u^2 = 3(5x)^2$
3. 相乘：$3(5x)^2 \times 5 = 15(5x)^2 = 375x^2$
恭喜！如果你理解了这个，你就掌握了链式法则！

记住：
🔍 像放大镜叠加 = 倍数相乘
⚙️ 像齿轮传动 = 比例相乘
📊 像斜率组合 = 斜率相乘

永远是相乘！
