选择案例

$x_n = n$
当 $n$ 无限增大时，$x_n$ 逐渐增大，没有一个固定的值。

数学解释

数列的项依次为：$1, 2, 3, 4, 5, \ldots, n, \ldots$
显然，随着 $n$ 的增加，数列中的每一项越来越大。
极限不存在

案例一：无界增长

$x_n = n$
特征分析：
• 单调递增
• 无上界
• 发散数列
• $\lim\limits_{n \to \infty} n = +\infty$

案例二：收敛到1

$x_n = \frac{n}{n+1}$
特征分析：
• 单调递增
• 有上界（上界为1）
• 收敛数列
• $\lim\limits_{n \to \infty} \frac{n}{n+1} = 1$

案例三：振荡数列

$x_n = \frac{1}{2}[1+(-1)^n]$
特征分析：
• 周期振荡
• 在0和1之间跳动
• 无极限
• 子序列有不同极限

案例四：交替变化趋向0

$x_n = \frac{1}{(-3)^n}$
特征分析：
• 正负交替变化
• 数值越来越小
• 最终趋向于0
• 例如：-0.33, 0.11, -0.037, 0.012...

极限判定准则

判断数列极限的简单方法
判断步骤：
1. 观察数列变化趋势（越来越大、越来越小、还是跳动）
2. 看数列是否有固定的"目标"（极限值）
3. 检查数列是否越来越接近某个数
4. 如果接近，那就是有极限；如果越来越大或跳动，就是没有极限
简单记忆：
有极限 = 数列有"家"可回
无极限 = 数列"流浪"或"跳动"
