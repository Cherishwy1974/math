前导案例：人口增长模型

问题：

某城市的规划部门需要预测该市人口变化。已知该城市在2024年1月1日的人口是100万人，根据历史数据分析，这个城市的人口增长率不恒定 — 每年新增的人口数量总是当前人口数量的5%。

规划部门提出问题：2024年8月1日的人口约是多少？

解决方法探索：

简单线性估算：

七个月的增长率：5% × (7/12) ≈ 2.917%

8月1日人口 = 100万 + (100万 × 2.917%) = 102.917万

线性估算的问题：

1月份增加的新人口，在2月份也会产生增长

2月份的新增人口，在3月份又会产生新的增长

实际的增长应该比102.917万要多一些

更细致的计算：

按月计算：

  • 1月末: 100万 + (100万 × 5% ÷ 12) = 100.417万
  • 2月末: 100.417万 + (100.417万 × 5% ÷ 12) = 100.835万
  • 3月末: 100.835万 + (100.835万 × 5% ÷ 12) = 101.255万
  • ...

甚至可以按天计算...

我们发现:

  • 计算问题繁琐、结果越精确
  • 实际情况是连续变化的，每时每刻都在发生
  • 这种"变化率与当前值相关"的关系，需要一种特殊的数学方程来描述

人口增长模型：微分方程解法

微分方程模型

我们尝试从基本的人口增长模型开始，建立一个城市的人口增长率与当前人口的关系。当我们把时间间隔 $\Delta t$ 无限细分，这种情况就可用一阶微分方程来描述：

$$\frac{dy}{dt} = ky \quad\text{或写作} \quad y' = ky$$

其中：

  • $y$ 表示人口数量（万人）
  • $t$ 表示时间（年）
  • $k$ 表示增长率系数（在这里 $k = 0.05$，即5%）

这个方程精确描述了人口增长的连续过程：任意时刻的增长率等于当时的人口数量乘以增长率系数。

微分方程求解

解得的指数增长模型：$y = y_0e^{kt}$

其中：

  • $y_0 = 100$（初始人口100万）
  • $k = 0.05$（年增长率5%）
  • $t = 0.5820$（从1月1日到8月1日的时间，计算方法：213/366年）
  • $e$ 是自然对数的底数，约等于2.71828

代入数值计算：$y = 100 \times e^{0.05×0.5820} \approx 102.94$ 万人

这说明2024年8月1日的人口约为102.94万人，比简单的线性计算（102.917万）略多一些。这些差异反映了精确的连续性：新增人口立即参与到下一时刻的增长过程中。这就是为什么我们需要用微分方程来描述这种连续变化的过程。

第一节 微分方程的基本概念

1.1 微分方程的概念

让我们理解方程的本质区别。普通方程描述的是变量之间的固定关系，比如 $x + 2 = 5$。而微分方程则包含未知函数及其导数，比如 $\frac{dN}{dt} = rN$ 表达了未知函数 $N(t)$ 与其导数 $\frac{dN}{dt}$ 之间的数学关系。

微分方程研究的核心问题是：已知一个函数的导数满足某种关系，求解出这个函数本身。例如从人口增长模型 $\frac{dN}{dt} = 0.05N$ 为例，等式左边 $\frac{dN}{dt}$ 是导数，右边 $0.05N$ 是相关的函数，我们提供了一种从导数方向寻找解决方案的思路。

这种通过导数关系确定函数的方法，为我们研究各类实际问题提供了重要工具。它不仅适用于时间过程（如人口增长），也可以描述空间分布（如温度场）等各种数学关系。微分方程的基本任务是：根据已知的导数关系，求解满足这些关系的函数。

微分方程定义

微分方程是表示未知函数及其导数与自变量之间关系的方程。方程中必须包含未知函数的导数，可以包含自变量和未知函数（但不是必须的）。

根据未知函数的性质，微分方程可分为：

  • 常微分方程（未知函数是一元函数）
  • 偏微分方程（未知函数是多元函数）

在此基础上，我们主要研究常微分方程，因此除特殊说明外，本章所说的微分方程均指常微分方程。

1.1.2 微分方程的阶

微分方程中所含未知函数的导数的最高阶为微分方程的阶。

以下是几个例子来理解：

方程 $\frac{dy}{dx} + xy = \sin x$ 中，最高阶导数是一阶导数 $\frac{dy}{dx}$，因此它是一阶微分方程

方程 $\frac{d^2y}{dx^2} + \frac{dy}{dx} - y = e^x$ 中，最高阶导数是二阶导数 $\frac{d^2y}{dx^2}$，所以它是二阶微分方程

方程 $\frac{d^3y}{dx^3} - \frac{d^2y}{dx^2} + (\frac{dy}{dx})^2 = 0$ 含有三阶导数 $\frac{d^3y}{dx^3}$，因此它是三阶微分方程

一般地，n阶微分方程中可能同时包含n阶、n-1阶直至一阶导数，也可能只包含最高的n阶导数。需要注意的是，微分方程的阶数仅取决于最高阶导数，而与其他低阶导数的存在与否无关。

1.1.3 微分方程的解

如果一个函数 $y = f(x)$ 代入微分方程后，使方程成为恒等式，那我们称 $y = f(x)$ 是这个微分方程的解。

例题：

当函数 $y = x^2$ 代入方程 $\frac{dy}{dx} = 2x$ 时，左边 $\frac{dy}{dx} = 2x$，右边也是 $2x$，等式成立，因此 $y = x^2$ 是这个微分方程的解。

计算 $y = x^2$ 的导数: $\frac{dy}{dx} = 2x$
代入原方程: $\frac{dy}{dx} = 2x$ 变成 $2x = 2x$
等式两边相等，恒等式成立，所以 $y = x^2$ 是原方程的解

求解过程中我们可以使用两种形式的解：

  • 显式解：直接表示 $y = f(x)$ 的形式
  • 隐函数解：以 $F(x,y) = 0$ 的形式表达，也称为参数方程形式

本质上，解微分方程就是从导数关系转化为函数关系。

通解与特解

通解：如果微分方程的解中含有任意常数，且任意常数的个数等于微分方程的阶数，这样的解称为微分方程的通解。

特解：通解中的任意常数取为确定值后得到的解称为特解。通过初始条件可以确定这些常数，从而得到唯一特解。

通解与特解示例

例：考虑方程 $\frac{dy}{dx} = 2x$

经验证：

  • $y = x^2 + 1$ 是它的一个解，因为代入后左边 $\frac{d(x^2+1)}{dx} = 2x$，等于右边 $2x$
  • $y = x^2 + 2$ 也是它的解，同理
  • $y = x^2 + 3$ 同样是这个方程的解

实际上，对于任意常数 $C$，$y = x^2 + C$ 都是该方程的解，因为左边 $\frac{d(x^2+C)}{dx} = 2x$，等于右边 $2x$

这说明这个微分方程有无数个解，而且这些解可以用含有一个任意常数 $C$ 的式子 $y = x^2 + C$ 表示。由于该方程是一阶微分方程，其通解中包含一个任意常数，所以 $y = x^2 + C$ 就是该微分方程的通解。

对于函数 $y = x^2 + C$，计算导数: $\frac{dy}{dx} = 2x$
代入方程 $\frac{dy}{dx} = 2x$，有 $2x = 2x$，恒等式成立
所以对任意常数 $C$，$y = x^2 + C$ 都是方程的解
这个一阶微分方程的通解中含有一个任意常数，即 $y = x^2 + C$
确定特解

通解表示了所有可能的解，而特解是通解中取特定常数值得到的解。例如：

  • $y = x^2 + 1$ 是 $C = 1$ 时的特解
  • $y = x^2 + 2$ 是 $C = 2$ 时的特解
  • $y = x^2 + 3$ 是 $C = 3$ 时的特解

如何确定我们要的特解？这就需要初始条件：

  • 若给定初始条件 $y|_{x=0} = 1$，代入通解 $1 = 0^2 + C$，所以 $C = 1$，特解为 $y = x^2 + 1$
  • 若给定初始条件 $y|_{x=0} = 2$，代入通解 $2 = 0^2 + C$，所以 $C = 2$，特解为 $y = x^2 + 2$
  • 若给定初始条件 $y|_{x=0} = 3$，代入通解 $3 = 0^2 + C$，所以 $C = 3$，特解为 $y = x^2 + 3$
练习：找出方程 $\frac{dy}{dx} = 2x$ 满足初始条件 $y|_{x=1} = 4$ 的特解
方程的通解为 $y = x^2 + C$
代入初始条件 $y|_{x=1} = 4$: $4 = 1^2 + C$
解得: $C = 3$
所以特解为: $y = x^2 + 3$

通解中任意常数个数与方程阶数的关系

在微分方程中，通解中任意常数的个数与方程阶数有密切关系：

  • 一阶方程需要一次积分，得到一个任意常数
  • 二阶方程需要两次积分，得到两个任意常数
  • 三阶方程需要三次积分，得到三个任意常数

这个规律告诉我们：通解中任意常数的个数等于微分方程的阶数，而通解中的任意常数与自变量的个数无关。

思考
填空题

在微分方程中，通解中的任意常数个数与方程的阶数相等，通过对应阶次的积分过程，我们发现一阶方程通过一次积分得到一个常数，二阶方程通过两次积分得到两个常数，三阶方程通过三次积分得到三个常数，因此，可以推广出n阶方程需要积分n次，得到n个任意常数。这就解释了为什么通解中任意常数个数必须等于微分方程的阶数：每做一次积分就会引入一个新的任意常数，而微分方程程的阶数好好对应着我们需要做的积分次数。

例题与习题

例1：指出下列方程中哪些是微分方程，并指出它们的阶数

$$x^2 + y^2 = 4$$

$$\frac{dy}{dx} + 3y = e^x$$

$$(\frac{dy}{dx})^2 + y = \sin x$$

$x^2 + y^2 = 4$ 不是微分方程，因为它不含有导数。
$\frac{dy}{dx} + 3y = e^x$ 是一阶微分方程，因为最高阶导数是一阶导数 $\frac{dy}{dx}$。
$(\frac{dy}{dx})^2 + y = \sin x$ 是一阶微分方程，因为最高阶导数是一阶导数 $\frac{dy}{dx}$。
例2：验证 $y = e^{2x}$ 是否为方程 $y'' - 4y = 0$ 的解
计算 $y = e^{2x}$ 的一阶导数：$y' = 2e^{2x}$
计算二阶导数：$y'' = 4e^{2x}$
代入原方程：$y'' - 4y = 4e^{2x} - 4e^{2x} = 0$
等式成立，因此 $y = e^{2x}$ 是方程 $y'' - 4y = 0$ 的解。
例3：验证 $y = e^x$ 是否为方程 $y' + 2y = 3e^x$ 的解
计算 $y = e^x$ 的导数：$y' = e^x$
代入原方程：$y' + 2y = e^x + 2e^x = 3e^x$
等式成立，因此 $y = e^x$ 是方程 $y' + 2y = 3e^x$ 的解。

例题与习题（续）

例4：验证函数 $y = \sin x$ 是否为微分方程 $y'' + y = 0$ 的解
计算 $y = \sin x$ 的一阶导数：$y' = \cos x$
计算二阶导数：$y'' = -\sin x$
代入原方程：$y'' + y = -\sin x + \sin x = 0$
等式成立，因此 $y = \sin x$ 是方程 $y'' + y = 0$ 的解。
例5：已知函数 $y = C_1e^x + C_2e^{-x}$ 是微分方程 $y'' - y = 0$ 的通解，求满足初始条件 $y|_{x=0} = 3$ 及 $y'|_{x=0} = 1$ 的特解
计算 $y = C_1e^x + C_2e^{-x}$ 的导数：$y' = C_1e^x - C_2e^{-x}$
代入初始条件 $y|_{x=0} = 3$：$C_1e^0 + C_2e^0 = C_1 + C_2 = 3$ ... (1)
代入初始条件 $y'|_{x=0} = 1$：$C_1e^0 - C_2e^0 = C_1 - C_2 = 1$ ... (2)
解方程组 (1) 和 (2)：
(1) + (2): $2C_1 = 4$，得 $C_1 = 2$
(1) - (2): $2C_2 = 2$，得 $C_2 = 1$
特解为：$y = 2e^x + e^{-x}$
例6：已知函数 $y = C_1\cos 2x + C_2\sin 2x$ 是微分方程 $y'' + 4y = 0$ 的通解，求满足初始条件 $y|_{x=0} = 1$ 及 $y'|_{x=0} = 2$ 的特解
计算 $y = C_1\cos 2x + C_2\sin 2x$ 的导数：$y' = -2C_1\sin 2x + 2C_2\cos 2x$
代入初始条件 $y|_{x=0} = 1$：$C_1\cos 0 + C_2\sin 0 = C_1 = 1$ ... (1)
代入初始条件 $y'|_{x=0} = 2$：$-2C_1\sin 0 + 2C_2\cos 0 = 2C_2 = 2$ ... (2)
由 (1) 和 (2) 得：$C_1 = 1$, $C_2 = 1$
特解为：$y = \cos 2x + \sin 2x$

1.2 可分离变量的微分方程

可分离变量的微分方程形式

可分离变量的微分方程具有以下形式：

$$\frac{dy}{dx} = f(x)g(y)$$

其中 $f(x)$ 只是 $x$ 的函数，$g(y)$ 只是 $y$ 的函数。

解法步骤
  1. 将方程化为标准形式 $\frac{dy}{dx} = f(x)g(y)$
  2. 注: 如果方程不是标准形式，需要通过移项、化简等将其转换为标准形式
    例如: $y' + 2xy = 0$ 转换为 $\frac{dy}{dx} = -2xy$
  3. 移项分离变量: $\frac{dy}{g(y)} = f(x)dx$
  4. 两边积分: $\int \frac{dy}{g(y)} = \int f(x)dx$
  5. 求解通解: $G(y) = F(x) + C$，其中 $C$ 为任意常数
例：求解方程 $\frac{dy}{dx} = xy$
将方程写为标准形式：$\frac{dy}{dx} = xy$，其中 $f(x) = x$，$g(y) = y$
分离变量：$\frac{dy}{y} = x dx$
两边积分：$\int \frac{dy}{y} = \int x dx$
计算积分：$\ln|y| = \frac{x^2}{2} + C$
求解 $y$：$y = e^{\frac{x^2}{2} + C} = e^C \cdot e^{\frac{x^2}{2}}$
令 $C_1 = e^C$（任意常数），得通解：$y = C_1e^{\frac{x^2}{2}}$

可分离变量的微分方程：例题

例1：求解微分方程 $\frac{dy}{dx} = 2x(y+1)$
将方程写为标准形式：$\frac{dy}{dx} = 2x(y+1)$，其中 $f(x) = 2x$，$g(y) = y+1$
分离变量：$\frac{dy}{y+1} = 2x dx$
两边积分：$\int \frac{dy}{y+1} = \int 2x dx$
计算积分：$\ln|y+1| = x^2 + C$
求解 $y$：$y+1 = e^{x^2+C} = e^C \cdot e^{x^2}$
令 $A = e^C$（任意常数），得通解：$y = Ae^{x^2} - 1$
例2：求解微分方程 $\frac{dy}{dx} = \frac{x}{y}$
将方程写为标准形式：$\frac{dy}{dx} = \frac{x}{y}$，其中 $f(x) = x$，$g(y) = \frac{1}{y}$
分离变量：$y dy = x dx$
两边积分：$\int y dy = \int x dx$
计算积分：$\frac{y^2}{2} = \frac{x^2}{2} + C$
通解：$y^2 = x^2 + 2C$
令 $C_1 = 2C$（任意常数），得通解：$y^2 = x^2 + C_1$

可分离变量的微分方程：更多例题

例3：求解微分方程 $\frac{dy}{dx} = 3x^2y$
将方程写为标准形式：$\frac{dy}{dx} = 3x^2y$，其中 $f(x) = 3x^2$，$g(y) = y$
分离变量：$\frac{dy}{y} = 3x^2 dx$
两边积分：$\int \frac{dy}{y} = \int 3x^2 dx$
计算积分：$\ln|y| = x^3 + C$
求解 $y$：$y = \pm e^{x^3+C} = \pm e^C \cdot e^{x^3}$
令 $A = \pm e^C$（任意常数），得通解：$y = Ae^{x^3}$
例4：求解微分方程 $\frac{dy}{dx} = \frac{y}{x+1}$
将方程写为标准形式：$\frac{dy}{dx} = \frac{y}{x+1}$，其中 $f(x) = \frac{1}{x+1}$，$g(y) = y$
分离变量：$\frac{dy}{y} = \frac{dx}{x+1}$
两边积分：$\int \frac{dy}{y} = \int \frac{dx}{x+1}$
计算积分：$\ln|y| = \ln|x+1| + C$
化简：$\ln|y| - \ln|x+1| = C$
进一步化简：$\ln|\frac{y}{x+1}| = C$
求解 $y$：$\frac{y}{x+1} = e^C$
令 $A = e^C$（任意常数），得通解：$y = A(x+1)$

1.3 齐次型微分方程

齐次函数

如果函数 $f(x,y)$ 满足 $f(tx,ty) = t^n f(x,y)$，其中 $t$ 为任意非零常数，$n$ 为常数，则称 $f(x,y)$ 为 $n$ 阶齐次函数。

例如:

  • $f(x,y) = \frac{y}{x}$ 是零阶齐次函数，因为 $f(tx,ty) = \frac{ty}{tx} = \frac{y}{x} = f(x,y)$
  • $f(x,y) = x^2 + xy + y^2$ 是二阶齐次函数，因为 $f(tx,ty) = t^2x^2 + t^2xy + t^2y^2 = t^2f(x,y)$
齐次型微分方程的形式

形如 $\frac{dy}{dx} = f(\frac{y}{x})$ 的微分方程称为齐次型微分方程。

求解步骤
  1. 判断是否为齐次方程：检查方程是否可以写成 $\frac{dy}{dx} = f(\frac{y}{x})$ 的形式
  2. 引入代换：
    • 令 $u = \frac{y}{x}$，则 $y = ux$
    • 求 $\frac{dy}{dx} = u + x\frac{du}{dx}$ (使用求导的乘法法则)
  3. 代入原方程：
    • 将 $y$ 用 $ux$ 代替
    • 将 $\frac{dy}{dx}$ 用 $u + x\frac{du}{dx}$ 代替
  4. 化简并分离变量：
    • 整理得关于 $u$ 的可分离变量方程
    • 分离变量并积分
  5. 回代还原：
    • 将 $u = \frac{y}{x}$ 代回
    • 得到原方程的通解

齐次型微分方程：例题

例1：求解微分方程 $\frac{dy}{dx} = \frac{x^2+y^2}{xy}$
检查是否为齐次方程：$\frac{dy}{dx} = \frac{x^2+y^2}{xy}$
将右边变形：$\frac{x^2+y^2}{xy} = \frac{x^2}{xy} + \frac{y^2}{xy} = \frac{x}{y} + \frac{y}{x}$
这是一个齐次方程，因为右边可以表示为 $\frac{y}{x}$ 的函数
令 $u = \frac{y}{x}$，则 $y = ux$
求导：$\frac{dy}{dx} = u + x\frac{du}{dx}$
代入原方程：$u + x\frac{du}{dx} = \frac{1+u^2}{u}$
整理：$x\frac{du}{dx} = \frac{1+u^2}{u} - u = \frac{1+u^2-u^2}{u} = \frac{1}{u}$
分离变量：$u du = \frac{dx}{x}$
两边积分：$\int u du = \int \frac{dx}{x}$
计算积分：$\frac{u^2}{2} = \ln|x| + C$
回代 $u = \frac{y}{x}$：$\frac{1}{2}(\frac{y}{x})^2 = \ln|x| + C$
化简：$y^2 = 2x^2(\ln|x| + C)$

齐次型微分方程：更多例题

例2：求解微分方程 $\frac{dy}{dx} = \frac{2x+y}{x+2y}$
检查是否为齐次方程：$\frac{dy}{dx} = \frac{2x+y}{x+2y}$
分子分母同时除以 $x$：$\frac{dy}{dx} = \frac{2+\frac{y}{x}}{1+2\frac{y}{x}}$
右边是 $\frac{y}{x}$ 的函数，所以这是一个齐次方程
令 $u = \frac{y}{x}$，则 $y = ux$
求导：$\frac{dy}{dx} = u + x\frac{du}{dx}$
代入原方程：$u + x\frac{du}{dx} = \frac{2+u}{1+2u}$
整理：$x\frac{du}{dx} = \frac{2+u}{1+2u} - u = \frac{2+u-u-2u^2}{1+2u} = \frac{2-2u^2}{1+2u}$
分离变量：$\frac{1+2u}{2-2u^2}du = \frac{dx}{x}$
化简左边：$\frac{1+2u}{2(1-u^2)}du = \frac{dx}{x}$
分解分母：$\frac{1+2u}{2(1-u)(1+u)}du = \frac{dx}{x}$
使用部分分式分解：$\frac{1+2u}{2(1-u)(1+u)} = \frac{A}{1-u} + \frac{B}{1+u}$
求解 $A$, $B$：$\frac{1+2u}{2(1-u)(1+u)} = \frac{1}{2(1-u)} + \frac{1/2+u}{2(1+u)}$
两边积分...(此处过程较复杂)...
最终得到：$\ln|x| + \ln|1-\frac{y}{x}| - \ln|1+\frac{y}{x}| = C$
化简：$\frac{x(1-\frac{y}{x})}{1+\frac{y}{x}} = e^C = C_1$
进一步化简：$\frac{x-y}{x+y} = C_1$

第二节 一阶线性微分方程

2.1 一阶线性齐次微分方程

微分方程中"线性"的概念

微分方程中的"线性"是指：方程对于因变量 $y$ 及其各阶导数 $y'$ 都是一次的（即线性的），不含有 $y$ 的高次项、乘积项或者函数项等。

例如，以下形式不能出现在线性微分方程中：

  • $y \cdot y'$ 不能出现（因为是乘积项）
  • $\sin y$ 不能出现（因为是函数项）
  • $(y')^2$ 不能出现（因为是高次项）

一阶线性齐次微分方程的标准形式

一阶线性齐次微分方程的标准形式是 $\frac{dy}{dx} + P(x)y = 0$ 或 $y' + P(x)y = 0$，其中 $P(x)$ 是 $x$ 的连续函数。

一阶线性齐次微分方程的特点:

  • 方程是线性的，即 $y$ 和 $\frac{dy}{dx}$ 的次数均为 1
  • 方程是齐次的，即方程右侧为 0

一阶线性齐次微分方程的解法步骤:

  1. 将方程化为标准形式 $\frac{dy}{dx} + P(x)y = 0$
  2. 移项变形: $\frac{dy}{y} = -P(x)dx$
  3. 两边积分: $\ln |y| = -\int P(x)dx + C$
  4. 解出y: $y = Ce^{-\int P(x)dx}$，其中 $C$ 为任意常数

一阶线性齐次微分方程：例题

例1：求解微分方程 $\frac{dy}{dx} + 3y = 0$
方程已是标准形式：$\frac{dy}{dx} + 3y = 0$，其中 $P(x) = 3$
移项变形：$\frac{dy}{y} = -3dx$
两边积分：$\int \frac{dy}{y} = -\int 3dx$
计算积分：$\ln|y| = -3x + C$
求解 $y$：$y = e^{-3x+C} = e^C \cdot e^{-3x}$
令 $C_1 = e^C$（任意常数），得通解：$y = C_1e^{-3x}$
例2：求解微分方程 $xy' + 2y = 0$
将方程改写为标准形式：$y' + \frac{2}{x}y = 0$，其中 $P(x) = \frac{2}{x}$
移项变形：$\frac{dy}{y} = -\frac{2}{x}dx$
两边积分：$\int \frac{dy}{y} = -\int \frac{2}{x}dx$
计算积分：$\ln|y| = -2\ln|x| + C$
化简：$\ln|y| = \ln|x^{-2}| + C = \ln|\frac{1}{x^2}| + C$
求解 $y$：$y = \frac{e^C}{x^2} = \frac{C_1}{x^2}$，其中 $C_1 = e^C$

2.2 一阶线性非齐次微分方程

一阶线性非齐次微分方程的标准形式

一阶线性非齐次微分方程的标准形式为:

$$\frac{dy}{dx} + P(x)y = Q(x) \text{ 或 } y' + P(x)y = Q(x)$$

其中 $P(x)$ 和 $Q(x)$ 是 $x$ 的连续函数，且 $Q(x) \neq 0$。

一阶线性非齐次微分方程求通解

公式法：$y = e^{-\int P(x)dx} \left(\int Q(x)e^{\int P(x)dx}dx + C\right)$

求齐次方程 $y' + P(x)y = 0$ 的通解 $y_c = Ce^{-\int P(x)dx}$

常数变易法：

  1. 化为标准形式，确定 $P(x)$ 和 $Q(x)$
  2. 计算积分 $\int P(x)dx$ 和 $e^{\int P(x)dx}$
  3. 计算 $Q(x)e^{\int P(x)dx}$ 的积分
  4. 代入公式得通解

一阶线性非齐次微分方程的通解为对应齐次方程的通解加上非齐次方程的一个特解：$y = y_c + y_p$。

一阶线性非齐次微分方程：例题

例1：求解微分方程 $\frac{dy}{dx} + y = e^{2x}$
方程已是标准形式：$\frac{dy}{dx} + y = e^{2x}$，其中 $P(x) = 1$，$Q(x) = e^{2x}$
计算 $\int P(x)dx = \int 1dx = x$
计算 $e^{\int P(x)dx} = e^x$
计算 $Q(x)e^{\int P(x)dx} = e^{2x} \cdot e^x = e^{3x}$
计算 $\int Q(x)e^{\int P(x)dx}dx = \int e^{3x}dx = \frac{1}{3}e^{3x}$
代入公式：$y = e^{-\int P(x)dx} \left(\int Q(x)e^{\int P(x)dx}dx + C\right) = e^{-x} \left(\frac{1}{3}e^{3x} + C\right)$
化简：$y = \frac{1}{3}e^{2x} + Ce^{-x}$
例2：求解微分方程 $y' + 2y = x$
方程已是标准形式：$y' + 2y = x$，其中 $P(x) = 2$，$Q(x) = x$
计算 $\int P(x)dx = \int 2dx = 2x$
计算 $e^{\int P(x)dx} = e^{2x}$
计算 $Q(x)e^{\int P(x)dx} = x \cdot e^{2x}$
计算 $\int Q(x)e^{\int P(x)dx}dx = \int xe^{2x}dx$
使用分部积分：$\int xe^{2x}dx = \frac{x}{2}e^{2x} - \frac{1}{4}e^{2x}$
代入公式：$y = e^{-2x} \left(\frac{x}{2}e^{2x} - \frac{1}{4}e^{2x} + C\right)$
化简：$y = \frac{x}{2} - \frac{1}{4} + Ce^{-2x}$

一阶线性微分方程应用：温度冷却模型

问题背景与实际意义

牛顿冷却定律描述了物体与周围环境之间的热量交换过程。在工程热学、食品加工和化工生产等领域有广应用。本案例研究一杯初始温度为 90°C 的热咖啡，在 20°C 室温环境下的冷却过程，目标是预测 10 分钟后的温度。

理论基础

根据牛顿冷却定律，物体的冷却速率与其温度和环境温度之差成正比，可以建立一阶线性非齐次微分方程来描述这一过程。

微分方程的标准形式：$\frac{dT}{dt} + k(T - T_e) = 0$，其中 $T$ 为时间 $t$ 时刻的物体温度

通解的形式：$T(t) = T_h(t) + T_p(t)$，即齐次解与特解的和

建立数学模型

根据牛顿冷却定律，物体的冷却速率 $\frac{dT}{dt}$ 与温差 $T - T_{env}$ 成正比：$\frac{dT}{dt} = -k(T - T_{env})$

其中：$k$ 是冷却系数；$T_{env} = 20$ °C为环境温度。整理方程为标准形式：$\frac{dT}{dt} + kT = kT_{env}$

求解过程

使用公式法求解该微分方程。

  1. 确定积分因子：将方程写为标准形式 $\frac{dT}{dt} + kT = kT_{env}$，积分因子为 $\mu(t) = e^{kt}$
  2. 求解通解：$T(t) = e^{-kt} \left(\int \mu(t) \cdot kT_{env}dt + C\right) = e^{-kt} \left(\int e^{kt} \cdot kT_{env}dt + C\right)$
  3. 计算得到温度随时间的演化模型：$T(t) = T_{env} + (T_0 - T_{env})e^{-kt}$ 其中 $T_0 = 90$ °C为初始温度
结果分析

得到的模型显示温度变化是指数衰减的，随着温度的减小，冷却速度逐渐降低。具体地：

  • 10 分钟后的温度 $T(10)$ 可以通过公式计算
  • 当 $t$ 趋近于无穷大，温度 $T(t)$ 趋近于室温 20°C

第三节 二阶常系数线性微分方程

3.1 二阶常系数线性齐次微分方程

线性微分方程

在自然科学和工程问题中，遇到的微分方程通常都是线性方程，或者可简化为线性方程。

形如 $y'' + p(x)y' + q(x)y = f(x)$ 是一个二阶线性微分方程。其中，$p(x)$、$q(x)$ 及 $f(x)$ 都是关于自变量 $x$ 的函数。

当 $p(x)$、$q(x)$ 为常数时，方程 $y'' + py' + qy = f(x)$ 称为二阶常系数线性微分方程。其中，当 $f(x) = 0$ 时，对应的方程 $y'' + py' + qy = 0$ 称为二阶常系数齐次线性微分方程；当 $f(x) \neq 0$ 时，对应的方程称为二阶常系数非齐次线性微分方程。

二阶常系数齐次线性微分方程

若 $y_1$, $y_2$ 是二阶齐次线性微分方程 $y'' + py' + qy = 0$ 的两个特解且 $\frac{y_1}{y_2}$ 不为常数，则方程的通解为 $y = C_1y_1 + C_2y_2$ （$C_1$、$C_2$ 为常数）。

指数函数 $y = e^{rx}$ 的一、二阶导数分别为 $y' = re^{rx}$、$y'' = r^2e^{rx}$，每次求导都是乘以一个常系数 $r$，函数形式保持不变。

如果我们使用 $y = e^{rx}$ 来解二阶常系数齐次线性微分方程 $y'' + py' + qy = 0$，会发现：

  • 先求一阶导数：$y' = re^{rx}$
  • 再求二阶导数：$y'' = r^2e^{rx}$
  • 代入原方程：$r^2e^{rx} + pre^{rx} + qe^{rx} = 0$
  • 整理得到：$e^{rx}(r^2 + pr + q) = 0$

由于 $e^{rx} \neq 0$，必须有 $r^2 + pr + q = 0$。

我们把方程 $r^2 + pr + q = 0$ 称为微分方程 $y'' + py' + qy = 0$ 的特征方程，它的根 $r_{1,2}$ 称为该微分方程的特征值。

特征方程与特征根

由于特征方程 $r^2 + pr + q = 0$ 是一个二次方程，它的根为 $r_{1,2} = \frac{-p \pm \sqrt{p^2-4q}}{2}$，所以根据判别式 $\Delta = p^2-4q$ 的情况，可分为以下三种情况讨论 $y'' + py' + qy = 0$ 的通解：

  • 当 $p^2 - 4q > 0$ 时，$r_1$, $r_2$ 是不相等的两个实根；
  • 当 $p^2 - 4q = 0$ 时，$r_1 = r_2 = r$ 是相等的两个实根；
  • 当 $p^2 - 4q < 0$ 时，$r_1$, $r_2$ 是一对共轭复根。
情况一：不相等的两个实根

若 $r_1$, $r_2$ 是不相等的两个实根，则分别对应两个特解 $y_1 = e^{r_1x}$, $y_2 = e^{r_2x}$，且 $\frac{y_1}{y_2} = e^{(r_1-r_2)x}$ 不为常数。

因此，方程 $y'' + py' + qy = 0$ 的通解为 $y = C_1e^{r_1x} + C_2e^{r_2x}$ （$C_1$、$C_2$ 为常数）。

情况二：相等的两个实根

若特征方程 $r^2 + pr + q = 0$ 有重根 $r_1 = r_2 = r = -\frac{p}{2}$，则方程 $y'' + py' + qy = 0$ 的通解为：

$y = (C_1 + C_2x)e^{rx}$ （$C_1$、$C_2$ 为常数）

情况三：一对共轭复根

若特征根为一对共轭复数，设 $r_1 = \alpha + i\beta$, $r_2 = \alpha - i\beta$ （$\beta \neq 0$）。则方程 $y'' + py' + qy = 0$ 的通解为：

$y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$ （$C_1$、$C_2$ 为常数）

求解步骤总结
  1. 写出特征方程 $r^2 + pr + q = 0$；
  2. 求出特征根 $r_1$, $r_2$；
  3. 根据特征根的情况写出通解：
    • 两个不同实根：$y = C_1e^{r_1x} + C_2e^{r_2x}$
    • 两个相等实根：$y = (C_1 + C_2x)e^{rx}$
    • 一对共轭复根：$y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$

二阶常系数线性齐次微分方程：例题

例1：求解微分方程 $y'' - 3y' + 2y = 0$
写出特征方程：$r^2 - 3r + 2 = 0$
求解特征方程：$r = \frac{3 \pm \sqrt{9-8}}{2} = \frac{3 \pm 1}{2}$
得到两个不同的实根：$r_1 = 2$, $r_2 = 1$
写出通解：$y = C_1e^{2x} + C_2e^x$，其中 $C_1$, $C_2$ 为任意常数
例2：求解微分方程 $y'' + 4y' + 4y = 0$
写出特征方程：$r^2 + 4r + 4 = 0$
求解特征方程：$r = \frac{-4 \pm \sqrt{16-16}}{2} = \frac{-4 \pm 0}{2} = -2$
得到两个相等的实根（重根）：$r_1 = r_2 = -2$
写出通解：$y = (C_1 + C_2x)e^{-2x}$，其中 $C_1$, $C_2$ 为任意常数
例3：求解微分方程 $y'' + 4y = 0$
写出特征方程：$r^2 + 4 = 0$
求解特征方程：$r = \pm 2i$
得到一对共轭复根：$r_1 = 2i$, $r_2 = -2i$，其中 $\alpha = 0$, $\beta = 2$
写出通解：$y = e^{0 \cdot x}(C_1\cos 2x + C_2\sin 2x) = C_1\cos 2x + C_2\sin 2x$

二阶常系数线性齐次微分方程：更多例题

例5：求解微分方程 $y'' - 4y' + 5y = 0$
写出特征方程：$r^2 - 4r + 5 = 0$
求解特征方程：$r = \frac{4 \pm \sqrt{16-20}}{2} = \frac{4 \pm \sqrt{-4}}{2} = \frac{4 \pm 2i}{2} = 2 \pm i$
得到一对共轭复根：$r_1 = 2 + i$, $r_2 = 2 - i$，其中 $\alpha = 2$, $\beta = 1$
写出通解：$y = e^{2x}(C_1\cos x + C_2\sin x)$，其中 $C_1$, $C_2$ 为任意常数
例6：求解微分方程 $y'' + 6y' + 9y = 0$
写出特征方程：$r^2 + 6r + 9 = 0$
求解特征方程：$r = \frac{-6 \pm \sqrt{36-36}}{2} = \frac{-6 \pm 0}{2} = -3$
得到两个相等的实根（重根）：$r_1 = r_2 = -3$
写出通解：$y = (C_1 + C_2x)e^{-3x}$，其中 $C_1$, $C_2$ 为任意常数
例7：求解微分方程 $y'' + y' - 2y = 0$
写出特征方程：$r^2 + r - 2 = 0$
求解特征方程：$r = \frac{-1 \pm \sqrt{1+8}}{2} = \frac{-1 \pm 3}{2}$
得到两个不同的实根：$r_1 = 1$, $r_2 = -2$
写出通解：$y = C_1e^x + C_2e^{-2x}$，其中 $C_1$, $C_2$ 为任意常数

3.2 二阶常系数线性非齐次微分方程

二阶常系数非齐次线性微分方程的通解结构

设二阶常系数非齐次线性微分方程为：$y'' + py' + qy = f(x)$

若 $y^*$ 是非齐次方程的一个特解
$Y$ 是对应的齐次方程 $y'' + py' + qy = 0$ 的通解

则其通解为：$y = Y + y^*$

求二阶常系数非齐次线性微分方程的通解，只要求得对应的齐次方程的通解，再求出非齐次方程的一个特解即可。

二阶常系数非齐次线性微分方程特解 $y^*$ 的求法（待定系数法）

第一类：$f(x) = e^{\lambda x}P_m(x)$ 型

1. 基本形式：$f(x) = e^{\lambda x}P_m(x)$

其中：$\lambda$ 是常数，$P_m(x)$ 是 $x$ 的 $m$ 次多项式

2. 特解的形式：$y^* = x^k Q_m(x)e^{\lambda x}$

其中：$Q_m(x)$ 是与 $P_m(x)$ 同次的待定多项式
$k$ 的值取决于 $\lambda$ 是否为特征根时：

  • 0，当 $\lambda$ 不是特征根时
  • 1，当 $\lambda$ 是特征方程的单重根时
  • 2，当 $\lambda$ 是特征方程的二重根时

第二类：$f(x) = A\cos\omega x + B\sin\omega x$ 型

1. 基本形式：$f(x) = A\cos\omega x + B\sin\omega x$

其中：$A$, $B$, $\omega$ 均为常数

2. 特解的形式：$y^* = x^k(a\cos\omega x + b\sin\omega x)$

其中：$a$, $b$ 为待定常数
$k$ 的值取决于 $\pm\omega i$ 是否为特征根：

  • 0，当 $\pm\omega i$ 不是特征根时
  • 1，当 $\pm\omega i$ 是特征根时

二阶常系数线性非齐次微分方程：例题

例1：求解微分方程 $y'' - y = x^2 + 1$
先求对应齐次方程 $y'' - y = 0$ 的通解
特征方程：$r^2 - 1 = 0$，求得 $r_1 = 1$, $r_2 = -1$
齐次方程的通解：$Y = C_1e^x + C_2e^{-x}$
设非齐次方程的特解形式。由于 $f(x) = x^2 + 1$ 是二次多项式，且 $\lambda = 0$ 不是特征根，所以特解形式为：$y^* = Ax^2 + Bx + C$
求导：$(y^*)' = 2Ax + B$，$(y^*)'' = 2A$
代入原方程：$2A - (Ax^2 + Bx + C) = x^2 + 1$
整理并比较系数：
$x^2$ 项：$-A = 1$，得 $A = -1$
$x$ 项：$-B = 0$，得 $B = 0$
常数项：$2A - C = 1$，代入 $A = -1$ 得 $C = -3$
得到特解：$y^* = -x^2 - 3$
通解：$y = Y + y^* = C_1e^x + C_2e^{-x} - x^2 - 3$
例2：求解微分方程 $y'' + 4y = 2\cos 2x$
先求对应齐次方程 $y'' + 4y = 0$ 的通解
特征方程：$r^2 + 4 = 0$，求得 $r_{1,2} = \pm 2i$
齐次方程的通解：$Y = C_1\cos 2x + C_2\sin 2x$
设非齐次方程的特解形式。由于 $f(x) = 2\cos 2x$，且 $\pm 2i$ 是特征根，所以特解形式为：$y^* = x(a\cos 2x + b\sin 2x)$
代入原方程并解出 $a$, $b$ 的值...(计算过程较复杂)
得到特解：$y^* = \frac{1}{2}x\sin 2x$
通解：$y = Y + y^* = C_1\cos 2x + C_2\sin 2x + \frac{1}{2}x\sin 2x$

二阶常系数线性微分方程应用案例：弹簧振动模型

问题背景与实际意义

弹簧振动系统是一个典型的二阶常系数线性微分方程模型，在机械工程、建筑减震等领域有重要应用。本案例研究一个质量为 $m = 1$ kg的物体，连接在弹簧系数为 $k = 4$ N/m的弹簧上，在阻尼系数为 $c = 2$ N·s/m的阻尼力作用下的振动情况。

理论基础与数学模型

根据牛顿第二定律和胡克定律，系统的运动方程为：$m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = 0$

代入具体参数，得到：$\frac{d^2x}{dt^2} + 2\frac{dx}{dt} + 4x = 0$

这是一个二阶常系数齐次线性微分方程，其中：

  • $x(t)$ 表示物体的位移
  • $\frac{dx}{dt}$ 表示速度
  • $\frac{d^2x}{dt^2}$ 表示加速度
求解过程
写出特征方程：$r^2 + 2r + 4 = 0$
求特征方程的根：$r = \frac{-2\pm\sqrt{4-16}}{2} = \frac{-2\pm\sqrt{-12}}{2} = -1 \pm \sqrt{3}i$
由于得到一对共轭复根：$r_{1,2} = -1 \pm \sqrt{3}i$，根据理论，通解形式为： $x(t) = e^{-t}(C_1\cos\sqrt{3}t + C_2\sin\sqrt{3}t)$
若初始条件为 $x(0) = 1$, $\frac{dx}{dt}(0) = 0$，代入求得：$C_1 = 1$, $C_2 = \frac{1}{\sqrt{3}}$
特解为：$x(t) = e^{-t}(\cos\sqrt{3}t + \frac{1}{\sqrt{3}}\sin\sqrt{3}t)$
结果分析
  • 指数项 $e^{-t}$ 表明阻尼作用使振幅随时间衰减
  • 物体做阻尼振动，振动周期为 $\frac{2\pi}{\sqrt{3}}$
  • 系统最终会趋于平衡位置（$t \to \infty$ 时，$x \to 0$）

综合应用：化工产品冷却质量控制

产线综合示范模块 [化工技术大类·工业过程温度控制]

对接标准：
GB/T 27025-2019 化工生产过程参数测量与控制技术规范
GB/T 20438-2017 化工过程安全完整性要求
GB/T 26544-2011 树脂生产工艺技术要求

区域产业：江苏省兴辉新化工产业园

问题描述

某化工厂树脂生产需要对温度为 130°C 的树脂进行冷却处理。经研究部门的测定，物体的冷却率与温度和环境温度之差成正比，其数学表达式为：$\frac{dT}{dt} = -k(T - T_c)$ 这方程的解为：$T = T_c + (T_0 - T_c)e^{-kt}$ 其中 $T$ 是物体温度（°C），$t$ 是时间（分钟），$k$ 是冷却系数，$T_c$ 是环境温度，$T_0$ 是初始温度。

生产线目标是温度为 60°C, 冷却系数为 $k = 0.015$，环境温度为 25°C, 要求冷却时间不超过 2 小时。

任务
  1. 预测达到目标温度所需时间
  2. 确定合适的生产线传送带速度
  3. 分析提高冷却效率的可行性
解答：
任务1: 计算达到目标温度的时间
代入已知条件：$T_0 = 130$°C, $T_c = 25$°C, $k = 0.015$, 目标温度 $T = 60$°C
原方程：$T = T_c + (T_0 - T_c)e^{-kt}$
代入求解 $t$：$60 = 25 + (130 - 25)e^{-0.015t}$
化简：$35 = 105e^{-0.015t}$
两边除以105：$\frac{35}{105} = e^{-0.015t}$
取对数：$\ln(\frac{1}{3}) = -0.015t$
得到：$t = \frac{\ln 3}{0.015} \approx 73$ 分钟
任务2: 确定生产线传送带速度
冷却处理需要73分钟，假设冷却通道长度为 $L$ 米
传送带速度应为 $v = \frac{L}{73}$ 米/分钟
为确保安全边际，实际速度可设为 $v = \frac{L}{80}$ 米/分钟
任务3: 分析提高冷却效率的可行性
从方程 $T = T_c + (T_0 - T_c)e^{-kt}$ 可知，提高冷却效率可从以下几方面考虑：
1. 增大冷却系数 $k$：可通过增加冷风量、使用冷却液等方式实现
2. 降低环境温度 $T_c$：可通过加强环境制冷实现
3. 分段冷却：前段快速冷却，后段缓慢冷却，避免热应力导致产品损坏
