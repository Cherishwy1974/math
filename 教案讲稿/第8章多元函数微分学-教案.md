# 第八章《多元函数微分学》教案

## 教学项目

**教学项目** 多元函数微分学

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已掌握一元函数微分学的基本概念和计算方法，但对多元函数微分学的理解还不够深入。学生在多元函数的概念、偏导数、全微分、方向导数、梯度方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生理解多元函数的概念、掌握偏导数和全微分的计算方法。  
能力目标：培养学生利用多元函数微分学解决实际问题的能力。  
素质目标：提高学生的数学抽象思维和空间想象能力，在工程场景中合理运用多元函数微分学方法。

**教学重点** 多元函数的概念、偏导数的计算、全微分的概念和计算。

**教学难点及应对**  
难点：偏导数的几何意义理解、全微分的概念和应用。  
应对策略：通过具体的几何图形演示，分步骤讲解，辅以Lab8系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第8章多元函数微分学》、Lab8系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：讲解多元函数的概念和偏导数的计算方法。  
问答法：鼓励学生提问，通过问题引导学生思考和深入理解。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab8系列软件演示多元函数微分学过程。

**教学反思** 需要关注学生对多元函数概念的理解是否到位，如果发现学生存在困惑，应进行针对性的复习和讲解。同时，要注意培养学生通过多元函数微分学方法解决实际问题的能力，强化其应用意识。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |

| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 情境导入（10min） | 【教师】讲述多元函数在工程中的应用案例<br>在工程实践中，我们经常需要处理多个变量的问题，比如温度分布、压力分布、成本优化等。如何用数学方法来描述和处理这些多变量问题？<br>【学生】思考并讨论多元函数的实际应用<br>【教师】展示工程案例：温度分布问题<br>案例背景：某工厂车间的温度分布$T(x,y)$依赖于位置坐标$(x,y)$，如何分析温度的变化规律？<br>【学生】讨论分析思路，提出初步解决方案<br>【教师】板书"多元函数→偏导数→全微分→应用"主线 | 激发学生学习兴趣，建立多元函数与实际工程的联系 |
| 多元函数的概念（20min） | 【教师】讲解多元函数的定义<br>定义1 设$D$是$n$维空间$R^n$的一个子集，如果对于$D$中的每一个点$(x_1, x_2, \ldots, x_n)$，按照某种对应法则$f$，都有唯一确定的实数$z$与之对应，则称$f$是定义在$D$上的$n$元函数，记作<br>$z = f(x_1, x_2, \ldots, x_n)$<br>其中$(x_1, x_2, \ldots, x_n) \in D$称为自变量，$z$称为因变量。<br>【教师】讲解二元函数的定义<br>当$n = 2$时，二元函数$z = f(x,y)$表示一个曲面。<br>【教师】讲解多元函数的定义域<br>使函数有意义的自变量的取值范围称为函数的定义域。<br>【教师】讲解多元函数的几何意义<br>1. 一元函数：平面上的曲线<br>2. 二元函数：空间中的曲面<br>3. 三元函数：四维空间中的超曲面<br>【教师】使用Lab8-1展示多元函数的几何意义<br>【学生】观察多元函数的几何表示<br>例1 求函数$z = \sqrt{4 - x^2 - y^2}$的定义域。<br>解：要使函数有意义，必须$4 - x^2 - y^2 \geq 0$<br>即$x^2 + y^2 \leq 4$<br>所以定义域为$D = \{(x,y) | x^2 + y^2 \leq 4\}$<br>这是一个以原点为圆心，半径为2的圆。<br>例2 求函数$z = \ln(x + y)$的定义域。<br>解：要使函数有意义，必须$x + y > 0$<br>所以定义域为$D = \{(x,y) | x + y > 0\}$<br>这是直线$x + y = 0$上方的半平面。<br>【学生】完成多元函数概念练习 | 学习多元函数的基本概念 |
| 偏导数的概念（25min） | 【教师】讲解偏导数的定义<br>定义2 设函数$z = f(x,y)$在点$(x_0, y_0)$的某个邻域内有定义，如果极限<br>$\lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x}$<br>存在，则称此极限为函数$f(x,y)$在点$(x_0, y_0)$处对$x$的偏导数，记作<br>$f_x(x_0, y_0)$或$\frac{\partial f}{\partial x}\bigg|_{(x_0, y_0)}$<br>类似地，对$y$的偏导数定义为<br>$\lim_{\Delta y \to 0} \frac{f(x_0, y_0 + \Delta y) - f(x_0, y_0)}{\Delta y}$<br>记作$f_y(x_0, y_0)$或$\frac{\partial f}{\partial y}\bigg|_{(x_0, y_0)}$<br>【教师】讲解偏导数的几何意义<br>1. $f_x(x_0, y_0)$表示曲面$z = f(x,y)$在点$(x_0, y_0, f(x_0, y_0))$处沿$x$轴方向的切线斜率<br>2. $f_y(x_0, y_0)$表示曲面$z = f(x,y)$在点$(x_0, y_0, f(x_0, y_0))$处沿$y$轴方向的切线斜率<br>【教师】讲解偏导数的计算方法<br>计算偏导数时，将其他变量视为常数，只对指定变量求导。<br>【教师】使用Lab8-2展示偏导数的几何意义<br>【学生】观察偏导数的几何表示<br>例3 求函数$z = x^2 + 3xy + y^2$的偏导数。<br>解：$\frac{\partial z}{\partial x} = 2x + 3y$（把$y$当常数）<br>$\frac{\partial z}{\partial y} = 3x + 2y$（把$x$当常数）<br>例4 求函数$z = e^{xy}$的偏导数。<br>解：$\frac{\partial z}{\partial x} = ye^{xy}$（把$y$当常数）<br>$\frac{\partial z}{\partial y} = xe^{xy}$（把$x$当常数）<br>例5 求函数$z = \sin(xy)$的偏导数。<br>解：$\frac{\partial z}{\partial x} = y\cos(xy)$（把$y$当常数）<br>$\frac{\partial z}{\partial y} = x\cos(xy)$（把$x$当常数）<br>【学生】完成偏导数计算练习 | 掌握偏导数的概念和计算方法 |
| 全微分的概念（25min） | 【教师】讲解全微分的定义<br>定义3 设函数$z = f(x,y)$在点$(x_0, y_0)$的某个邻域内有定义，如果函数在点$(x_0, y_0)$处的全增量<br>$\Delta z = f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)$<br>可以表示为<br>$\Delta z = A\Delta x + B\Delta y + o(\rho)$<br>其中$A$、$B$是与$\Delta x$、$\Delta y$无关的常数，$\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$，$o(\rho)$是$\rho$的高阶无穷小，则称函数$f(x,y)$在点$(x_0, y_0)$处可微，$A\Delta x + B\Delta y$称为函数在点$(x_0, y_0)$处的全微分，记作<br>$dz = A\Delta x + B\Delta y$<br>【教师】讲解全微分的计算<br>如果函数$z = f(x,y)$在点$(x_0, y_0)$处可微，则<br>$dz = f_x(x_0, y_0)dx + f_y(x_0, y_0)dy$<br>【教师】讲解可微的条件<br>如果函数$z = f(x,y)$在点$(x_0, y_0)$处的偏导数$f_x(x_0, y_0)$和$f_y(x_0, y_0)$存在且连续，则函数在该点可微。<br>【教师】使用Lab8-3展示全微分的几何意义<br>【学生】观察全微分的几何表示<br>例6 求函数$z = x^2 + 3xy + y^2$在点$(1, 2)$处的全微分。<br>解：先求偏导数：<br>$f_x(x,y) = 2x + 3y$，$f_y(x,y) = 3x + 2y$<br>在点$(1, 2)$处：<br>$f_x(1, 2) = 2 \cdot 1 + 3 \cdot 2 = 8$<br>$f_y(1, 2) = 3 \cdot 1 + 2 \cdot 2 = 7$<br>所以$dz = 8dx + 7dy$<br>例7 求函数$z = e^{xy}$在点$(0, 1)$处的全微分。<br>解：先求偏导数：<br>$f_x(x,y) = ye^{xy}$，$f_y(x,y) = xe^{xy}$<br>在点$(0, 1)$处：<br>$f_x(0, 1) = 1 \cdot e^0 = 1$<br>$f_y(0, 1) = 0 \cdot e^0 = 0$<br>所以$dz = dx$<br>【学生】完成全微分计算练习 | 学习全微分的概念和计算方法 |
| 方向导数与梯度（15min） | 【教师】讲解方向导数的定义<br>定义4 设函数$z = f(x,y)$在点$P_0(x_0, y_0)$的某个邻域内有定义，$l$是从$P_0$出发的射线，$P(x, y)$是$l$上任意一点，如果极限<br>$\lim_{\rho \to 0} \frac{f(x, y) - f(x_0, y_0)}{\rho}$<br>存在，则称此极限为函数$f(x,y)$在点$P_0$处沿方向$l$的方向导数，记作$\frac{\partial f}{\partial l}\bigg|_{P_0}$<br>【教师】讲解梯度的定义<br>定义5 设函数$z = f(x,y)$在点$P_0(x_0, y_0)$处可微，则向量<br>$\nabla f(x_0, y_0) = (f_x(x_0, y_0), f_y(x_0, y_0))$<br>称为函数$f(x,y)$在点$P_0$处的梯度。<br>【教师】讲解方向导数的计算<br>如果函数$z = f(x,y)$在点$P_0(x_0, y_0)$处可微，则<br>$\frac{\partial f}{\partial l}\bigg|_{P_0} = f_x(x_0, y_0)\cos\alpha + f_y(x_0, y_0)\cos\beta$<br>其中$(\cos\alpha, \cos\beta)$是方向$l$的单位向量。<br>【教师】使用Lab8-4展示方向导数和梯度<br>【学生】观察方向导数和梯度的几何意义<br>例8 求函数$z = x^2 + y^2$在点$(1, 1)$处沿方向$(1, 1)$的方向导数。<br>解：先求偏导数：<br>$f_x(x,y) = 2x$，$f_y(x,y) = 2y$<br>在点$(1, 1)$处：<br>$f_x(1, 1) = 2$，$f_y(1, 1) = 2$<br>方向$(1, 1)$的单位向量为$(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}})$<br>所以$\frac{\partial f}{\partial l}\bigg|_{(1,1)} = 2 \cdot \frac{1}{\sqrt{2}} + 2 \cdot \frac{1}{\sqrt{2}} = 2\sqrt{2}$<br>【学生】完成方向导数练习 | 学习方向导数和梯度的概念 |
| 课堂测验（10min） | 【教师】出几道测试题目<br>1. 求函数$z = x^2 + 3xy + y^2$的偏导数<br>2. 求函数$z = e^{xy}$在点$(0, 1)$处的全微分<br>3. 求函数$z = \sin(xy)$的偏导数<br>4. 求函数$z = x^2 + y^2$在点$(1, 1)$处的梯度<br>【学生】做测试题目<br>【教师】公布答案并讲解<br>解1：$\frac{\partial z}{\partial x} = 2x + 3y$，$\frac{\partial z}{\partial y} = 3x + 2y$<br>解2：$dz = dx$<br>解3：$\frac{\partial z}{\partial x} = y\cos(xy)$，$\frac{\partial z}{\partial y} = x\cos(xy)$<br>解4：$\nabla f(1, 1) = (2, 2)$ | 通过测试，了解学生对知识点的掌握情况 |
| 课堂小结（8min） | 【教师】总结本节课要点<br>1. 多元函数的概念和定义域<br>2. 偏导数的概念和计算方法<br>3. 全微分的概念和计算<br>4. 方向导数和梯度的概念<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |

## 教学提示

- 鼓励学生截图Lab8模拟结果作为报告证据，提升数据说服力
- 引导学生在偏导数计算中注意变量处理
- 结合工程案例，让学生体验多元函数微分学在工程中的重要作用
- 强调多元函数微分学在优化问题、物理建模中的重要作用
- 通过实际工程案例，培养学生运用多元函数微分学方法解决实际问题的能力
