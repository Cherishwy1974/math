# 第十二章《向量代数与空间解析几何》第三节课教案

## 教学项目

**教学项目** 空间直线与曲面

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已掌握向量的基本运算和平面方程，但对空间直线方程和空间曲面的理解还不够深入。学生在空间直线方程的建立、空间曲面的识别和应用方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生理解空间直线方程、掌握常见空间曲面的方程和性质。  
能力目标：培养学生利用空间直线和曲面方程解决实际几何问题的能力。  
素质目标：提高学生的空间想象能力和几何分析意识，在工程场景中合理运用向量方法。

**教学重点** 空间直线方程、常见空间曲面的方程。

**教学难点及应对**  
难点：空间直线方程的建立、空间曲面的识别。  
应对策略：通过具体的几何图形演示，分步骤讲解，辅以Lab12系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第12章向量代数与空间解析几何》、Lab12系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：讲解空间直线方程、常见空间曲面的方程和性质。  
问答法：鼓励学生提问，通过问题引导学生思考和深入理解。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab12系列软件演示空间直线和曲面。

**教学反思** 需要关注学生对空间直线和曲面概念的理解是否到位，如果发现学生存在困惑，应进行针对性的复习和讲解。同时，要注意培养学生通过空间直线和曲面方程解决实际几何问题的能力，强化其应用意识。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |
| --- | --- | --- |
| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 复习回顾（8min） | 【教师】回顾上节课内容<br>1. 向量的数量积和向量积<br>2. 平面方程的建立<br>3. 向量积的几何意义<br>【学生】回答教师提问，巩固上节课知识<br>【教师】检查作业完成情况，解答学生疑问 | 巩固上节课所学知识，为新课做准备 |
| 空间直线方程（30min） | 【教师】讲解空间直线的点向式方程<br>定义9 过点$P_0(x_0, y_0, z_0)$且方向向量为$\vec{s} = (m, n, p)$的直线方程为：<br>$\frac{x - x_0}{m} = \frac{y - y_0}{n} = \frac{z - z_0}{p}$<br>【教师】讲解空间直线的参数式方程<br>$\begin{cases} x = x_0 + mt \\ y = y_0 + nt \\ z = z_0 + pt \end{cases}$<br>其中$t$为参数<br>【教师】讲解空间直线的一般式方程<br>直线可以表示为两个平面的交线：<br>$\begin{cases} A_1x + B_1y + C_1z + D_1 = 0 \\ A_2x + B_2y + C_2z + D_2 = 0 \end{cases}$<br>【教师】讲解直线与平面的位置关系<br>（1）直线在平面内：$\vec{s} \cdot \vec{n} = 0$且直线上一点在平面内<br>（2）直线与平面平行：$\vec{s} \cdot \vec{n} = 0$且直线上一点不在平面内<br>（3）直线与平面相交：$\vec{s} \cdot \vec{n} \neq 0$<br>【教师】使用Lab12-7展示空间直线方程<br>【学生】观察空间直线的几何意义<br>例16 求过点$P(1, 2, 3)$且方向向量为$\vec{s} = (2, -1, 3)$的直线方程。<br>解：点向式方程：$\frac{x - 1}{2} = \frac{y - 2}{-1} = \frac{z - 3}{3}$<br>参数式方程：$\begin{cases} x = 1 + 2t \\ y = 2 - t \\ z = 3 + 3t \end{cases}$<br>例17 求过两点$A(1, 2, 3)$和$B(4, 6, 9)$的直线方程。<br>解：方向向量：$\vec{s} = \vec{AB} = (4-1, 6-2, 9-3) = (3, 4, 6)$<br>点向式方程：$\frac{x - 1}{3} = \frac{y - 2}{4} = \frac{z - 3}{6}$<br>例18 判断直线$\frac{x - 1}{2} = \frac{y - 2}{-1} = \frac{z - 3}{3}$与平面$2x - y + 3z - 5 = 0$的位置关系。<br>解：直线的方向向量：$\vec{s} = (2, -1, 3)$<br>平面的法向量：$\vec{n} = (2, -1, 3)$<br>$\vec{s} \cdot \vec{n} = 2 \times 2 + (-1) \times (-1) + 3 \times 3 = 4 + 1 + 9 = 14 \neq 0$<br>因为$\vec{s} \cdot \vec{n} \neq 0$，所以直线与平面相交<br>【学生】完成空间直线方程练习 | 学习空间直线方程的建立方法 |
| 常见空间曲面（35min） | 【教师】讲解球面方程<br>定义10 以点$C(a, b, c)$为球心，半径为$R$的球面方程为：<br>$(x - a)^2 + (y - b)^2 + (z - c)^2 = R^2$<br>特别地，以原点为球心的球面方程为：$x^2 + y^2 + z^2 = R^2$<br>【教师】讲解圆柱面方程<br>（1）以z轴为轴的圆柱面：$x^2 + y^2 = R^2$<br>（2）以x轴为轴的圆柱面：$y^2 + z^2 = R^2$<br>（3）以y轴为轴的圆柱面：$x^2 + z^2 = R^2$<br>【教师】讲解椭球面方程<br>$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$<br>当$a = b = c$时，椭球面退化为球面<br>【教师】讲解抛物面方程<br>（1）椭圆抛物面：$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2z$<br>（2）双曲抛物面：$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2z$<br>【教师】讲解双曲面方程<br>（1）单叶双曲面：$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$<br>（2）双叶双曲面：$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = -1$<br>【教师】使用Lab12-8展示常见空间曲面<br>【学生】观察空间曲面的几何特征<br>例19 求以点$C(1, 2, 3)$为球心，半径为4的球面方程。<br>解：$(x - 1)^2 + (y - 2)^2 + (z - 3)^2 = 16$<br>例20 求以z轴为轴，半径为3的圆柱面方程。<br>解：$x^2 + y^2 = 9$<br>例21 求椭球面$\frac{x^2}{4} + \frac{y^2}{9} + \frac{z^2}{16} = 1$与坐标平面的交线。<br>解：与xOy平面（z=0）的交线：$\frac{x^2}{4} + \frac{y^2}{9} = 1$（椭圆）<br>与yOz平面（x=0）的交线：$\frac{y^2}{9} + \frac{z^2}{16} = 1$（椭圆）<br>与xOz平面（y=0）的交线：$\frac{x^2}{4} + \frac{z^2}{16} = 1$（椭圆）<br>例22 求抛物面$z = x^2 + y^2$与平面$z = 4$的交线。<br>解：联立方程：$\begin{cases} z = x^2 + y^2 \\ z = 4 \end{cases}$<br>得：$x^2 + y^2 = 4$（圆）<br>【学生】完成空间曲面练习 | 学习常见空间曲面的方程和性质 |
| 综合应用（15min） | 【教师】讲解工程应用案例<br>案例：某建筑物的屋顶设计<br>背景：某建筑物的屋顶采用抛物面设计，方程为$z = \frac{x^2}{4} + \frac{y^2}{4}$，高度限制为$z \leq 9$<br>问题1：求屋顶的边界曲线方程<br>问题2：求屋顶在xOy平面上的投影面积<br>问题3：求屋顶的最大高度<br>【学生】分组讨论解决方案<br>解1：边界曲线：$\begin{cases} z = \frac{x^2}{4} + \frac{y^2}{4} \\ z = 9 \end{cases}$<br>即：$x^2 + y^2 = 36$（圆）<br>解2：投影面积 = $\pi \times 6^2 = 36\pi$<br>解3：最大高度 = 9<br>【教师】总结工程应用要点<br>1. 空间几何在建筑设计中的应用<br>2. 曲面方程在工程建模中的作用<br>3. 数学软件在几何分析中的应用<br>【学生】完成综合应用练习 | 学习空间几何在工程中的应用 |
| 课堂测验（10min） | 【教师】出几道测试题目<br>1. 求过点$(1, 2, 3)$且方向向量为$(2, -1, 3)$的直线方程<br>2. 求以原点为球心，半径为5的球面方程<br>3. 求椭球面$\frac{x^2}{9} + \frac{y^2}{4} + \frac{z^2}{16} = 1$与xOy平面的交线<br>4. 判断直线$\frac{x - 1}{2} = \frac{y - 2}{-1} = \frac{z - 3}{3}$与平面$2x - y + 3z - 5 = 0$的位置关系<br>【学生】做测试题目<br>【教师】公布答案并讲解<br>解1：$\frac{x - 1}{2} = \frac{y - 2}{-1} = \frac{z - 3}{3}$<br>解2：$x^2 + y^2 + z^2 = 25$<br>解3：$\frac{x^2}{9} + \frac{y^2}{4} = 1$（椭圆）<br>解4：相交（方向向量与法向量不垂直） | 通过测试，了解学生对知识点的掌握情况 |
| 课堂小结（8min） | 【教师】总结本节课要点<br>1. 空间直线方程的建立方法<br>2. 常见空间曲面的方程和性质<br>3. 空间几何在工程中的应用<br>4. 数学软件在几何分析中的作用<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |
