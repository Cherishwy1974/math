# 第十二章《向量代数与空间解析几何》第一节课教案

## 教学项目

**教学项目** 向量基础与运算

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已具备平面解析几何基础，但对空间概念和向量运算理解不够深入。学生在向量运算的几何意义、空间想象能力、向量积的计算和应用方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生理解空间直角坐标系、向量的概念、向量的表示方法、向量的线性运算。  
能力目标：培养学生利用向量运算解决实际几何问题的能力。  
素质目标：提高学生的空间想象能力和几何分析意识，在工程场景中合理运用向量方法。

**教学重点** 空间直角坐标系、向量的概念、向量的表示、向量的线性运算。

**教学难点及应对**  
难点：向量的几何意义理解、空间向量的运算。  
应对策略：通过具体的几何图形演示，分步骤讲解，辅以Lab12系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第12章向量代数与空间解析几何》、Lab12系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：讲解向量的基本概念、性质及其运算方法。  
问答法：鼓励学生提问，通过问题引导学生思考和深入理解。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab12系列软件演示向量运算过程。

**教学反思** 需要关注学生对向量几何意义的理解是否到位，如果发现学生存在困惑，应在下一课时进行针对性的复习和讲解。同时，要注意培养学生通过向量运算解决实际几何问题的能力，强化其应用意识。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |
| --- | --- | --- |
| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 情境导入（10min） | 【教师】讲述工程中的向量应用案例<br>在工程实践中，我们经常需要描述力、速度、位移等物理量。这些量不仅有大小，还有方向，如何用数学语言来描述？<br>【学生】思考并讨论向量的实际应用<br>【教师】展示工程案例：机械臂运动分析<br>案例背景：某工业机器人的机械臂需要从A点移动到B点，如何描述这个运动过程？<br>【学生】讨论分析思路，提出初步解决方案<br>【教师】板书"向量→运算→应用"主线 | 激发学生学习兴趣，建立向量与实际工程的联系 |
| 空间直角坐标系（20min） | 【教师】讲解空间直角坐标系<br>定义1 在空间中，过定点O作三条互相垂直的数轴，分别称为x轴、y轴、z轴，统称为坐标轴。三条坐标轴的交点O称为坐标原点，这样就建立了空间直角坐标系Oxyz。<br>【教师】讲解坐标轴的方向<br>右手法则：右手握住z轴，四指从x轴转向y轴，拇指指向z轴正方向<br>【教师】讲解点的坐标<br>空间中任意一点P，过P作三个坐标轴的垂线，垂足分别为A、B、C，则点P的坐标为P(x,y,z)，其中x、y、z分别是A、B、C在对应坐标轴上的坐标<br>【教师】演示Lab12-1，展示空间直角坐标系<br>【学生】观察空间直角坐标系的建立过程<br>例1 在空间直角坐标系中，标出点A(1,2,3)、B(-1,0,2)、C(0,3,-1)的位置。<br>解：根据坐标的定义，在坐标系中标出各点位置<br>例2 求点A(1,2,3)关于坐标平面的对称点。<br>解：关于xOy平面：(1,2,-3)<br>关于yOz平面：(-1,2,3)<br>关于xOz平面：(1,-2,3)<br>【学生】完成坐标系练习 | 学习空间直角坐标系，建立空间概念 |
| 向量的概念（25min） | 【教师】讲解向量的定义<br>定义2 既有大小又有方向的量称为向量。<br>向量 = 大小 + 方向<br>【教师】讲解向量的表示方法<br>（1）几何表示：用有向线段表示<br>（2）坐标表示：在空间内 $\vec{a} = (a_1, a_2, a_3)$<br>【教师】讲解向量的模<br>$|\vec{a}| = \sqrt{a_1^2 + a_2^2 + a_3^2}$<br>【教师】讲解单位向量<br>$\vec{e} = \frac{\vec{a}}{|\vec{a}|}$<br>【教师】讲解零向量和相等向量<br>零向量：模为0的向量，记作$\vec{0}$<br>相等向量：大小相等、方向相同的向量<br>【教师】演示Lab12-2，展示向量的几何表示<br>【学生】观察向量在不同坐标系中的表示<br>例3 已知向量$\vec{a} = (3, 4, 5)$，求$|\vec{a}|$和单位向量。<br>解：$|\vec{a}| = \sqrt{3^2 + 4^2 + 5^2} = \sqrt{50} = 5\sqrt{2}$<br>单位向量：$\vec{e} = \frac{1}{5\sqrt{2}}(3, 4, 5) = (\frac{3}{5\sqrt{2}}, \frac{4}{5\sqrt{2}}, \frac{5}{5\sqrt{2}})$<br>例4 已知点A(1, 2, 3)和B(4, 6, 9)，求向量$\vec{AB}$。<br>解：$\vec{AB} = (4-1, 6-2, 9-3) = (3, 4, 6)$<br>例5 已知向量$\vec{a} = (1, 2, 3)$，$\vec{b} = (2, 1, 4)$，求$|\vec{a}|$、$|\vec{b}|$和$|\vec{a} - \vec{b}|$。<br>解：$|\vec{a}| = \sqrt{1^2 + 2^2 + 3^2} = \sqrt{14}$<br>$|\vec{b}| = \sqrt{2^2 + 1^2 + 4^2} = \sqrt{21}$<br>$\vec{a} - \vec{b} = (1-2, 2-1, 3-4) = (-1, 1, -1)$<br>$|\vec{a} - \vec{b}| = \sqrt{(-1)^2 + 1^2 + (-1)^2} = \sqrt{3}$<br>【学生】完成向量表示练习 | 学习向量的基本概念，建立几何直观 |
| 向量的线性运算（25min） | 【教师】讲解向量加法<br>定义3 向量加法：$\vec{a} + \vec{b} = (a_1 + b_1, a_2 + b_2, a_3 + b_3)$<br>几何意义：平行四边形法则或三角形法则<br>【教师】讲解向量减法<br>定义4 向量减法：$\vec{a} - \vec{b} = (a_1 - b_1, a_2 - b_2, a_3 - b_3)$<br>几何意义：$\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$<br>【教师】讲解向量数乘<br>定义5 向量数乘：$k\vec{a} = (ka_1, ka_2, ka_3)$<br>几何意义：向量的伸缩和反向<br>【教师】讲解运算性质<br>（1）$\vec{a} + \vec{b} = \vec{b} + \vec{a}$（交换律）<br>（2）$(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})$（结合律）<br>（3）$k(\vec{a} + \vec{b}) = k\vec{a} + k\vec{b}$（分配律）<br>【教师】使用Lab12-3展示向量运算的几何意义<br>【学生】观察向量运算的动画演示<br>例6 已知$\vec{a} = (1, 2, 3)$，$\vec{b} = (4, 5, 6)$，求$\vec{a} + \vec{b}$和$2\vec{a} - 3\vec{b}$。<br>解：$\vec{a} + \vec{b} = (1+4, 2+5, 3+6) = (5, 7, 9)$<br>$2\vec{a} - 3\vec{b} = 2(1, 2, 3) - 3(4, 5, 6) = (2, 4, 6) - (12, 15, 18) = (-10, -11, -12)$<br>例7 已知点A(1, 1, 1)，B(2, 3, 4)，C(0, 1, 2)，求向量$\vec{AB} + \vec{BC}$。<br>解：$\vec{AB} = (2-1, 3-1, 4-1) = (1, 2, 3)$<br>$\vec{BC} = (0-2, 1-3, 2-4) = (-2, -2, -2)$<br>$\vec{AB} + \vec{BC} = (1-2, 2-2, 3-2) = (-1, 0, 1)$<br>例8 某机械臂从原点O移动到点A(3, 4, 5)，再移动到点B(6, 8, 10)，求总位移向量。<br>解：总位移 = $\vec{OA} + \vec{AB} = (3, 4, 5) + (6-3, 8-4, 10-5) = (3, 4, 5) + (3, 4, 5) = (6, 8, 10)$<br>【学生】完成向量运算练习 | 掌握向量的基本运算规则，培养几何直观 |
| 课堂测验（10min） | 【教师】出几道测试题目<br>1. 已知向量$\vec{a} = (2, 3, 4)$，求$|\vec{a}|$<br>2. 已知$\vec{a} = (1, 2, 3)$，$\vec{b} = (4, 5, 6)$，求$\vec{a} + \vec{b}$<br>3. 已知点A(1, 2, 3)和B(4, 6, 9)，求向量$\vec{AB}$的模<br>4. 计算$3\vec{a} - 2\vec{b}$，其中$\vec{a} = (1, 0, 1)$，$\vec{b} = (0, 1, 1)$<br>【学生】做测试题目<br>【教师】公布答案并讲解<br>解1：$|\vec{a}| = \sqrt{2^2 + 3^2 + 4^2} = \sqrt{29}$<br>解2：$\vec{a} + \vec{b} = (1+4, 2+5, 3+6) = (5, 7, 9)$<br>解3：$|\vec{AB}| = \sqrt{(4-1)^2 + (6-2)^2 + (9-3)^2} = \sqrt{9+16+36} = \sqrt{61}$<br>解4：$3\vec{a} - 2\vec{b} = 3(1, 0, 1) - 2(0, 1, 1) = (3, 0, 3) - (0, 2, 2) = (3, -2, 1)$ | 通过测试，了解学生对知识点的掌握情况 |
| 课堂小结（8min） | 【教师】总结本节课要点<br>1. 空间直角坐标系的建立<br>2. 向量的概念和表示方法<br>3. 向量的线性运算（加法、减法、数乘）<br>4. 向量运算的几何意义<br>5. 工程应用案例<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |
