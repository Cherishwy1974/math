# 第一章《函数与初等函数》教案

## 教学项目

**教学项目** 函数与初等函数

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已具备基本的数学基础，但对函数概念的理解还不够深入。学生在函数的概念、性质、基本初等函数、复合函数方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生深入理解函数的概念、掌握函数的基本性质和基本初等函数、熟练运用复合函数。  
能力目标：培养学生利用函数解决实际问题的能力、提高数学建模能力。  
素质目标：提高学生的数学抽象思维和逻辑推理能力，培养严谨的数学态度。

**教学重点** 函数的概念、函数的基本性质、基本初等函数、复合函数的处理。

**教学难点及应对**  
难点：函数概念的理解、复合函数的处理、函数性质的证明。  
应对策略：通过具体的工程案例演示，分步骤详细讲解，辅以Lab1系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第1章函数与初等函数》、Lab1系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：详细讲解函数的概念和基本性质。  
问答法：鼓励学生提问，通过问题引导学生深入思考。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab1系列软件演示函数过程。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |
| --- | --- | --- |
| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 情境导入（15min） | 【教师】讲述函数在工程中的应用案例<br>在工程实践中，我们经常需要描述变量之间的关系，比如温度与时间、速度与时间、电流与电压等。如何用数学方法来解决这些问题？<br>【学生】思考并讨论函数的实际意义<br>【教师】展示工程案例：<br>1. 温度控制系统：T(t) = T₀ + (T₁ - T₀)e^(-kt)<br>2. 电路分析：I = V/R<br>3. 人口增长模型：P(t) = P₀e^(rt)<br>【学生】讨论分析思路，提出初步解决方案<br>【教师】板书"函数→建立模型→求解→应用"主线 | 激发学生学习兴趣，建立函数与实际工程的联系 |
| 函数的概念（30min） | 【教师】详细讲解函数的定义<br>**定义1** 设$D$是实数集$R$的一个子集，如果对于$D$中的每一个数$x$，按照某种对应法则$f$，都有唯一确定的实数$y$与之对应，则称$f$是定义在$D$上的函数，记作$y = f(x)$。<br>其中：<br>- $D$称为函数的定义域<br>- $f(D) = \{y | y = f(x), x \in D\}$称为函数的值域<br>- $x$称为自变量<br>- $y$称为因变量<br>【教师】讲解函数的表示方法<br>1. **解析法**：用数学表达式表示，如$y = x^2 + 2x + 1$<br>2. **图像法**：用图形表示，如平面上的曲线<br>3. **表格法**：用表格表示，如实验数据<br>4. **描述法**：用文字描述，如"温度随时间的变化"<br>【教师】使用Lab1-1展示函数的概念<br>【学生】观察函数的几何表示<br>**例1** 判断下列关系是否为函数：<br>（1）$y = x^2$<br>（2）$x^2 + y^2 = 1$<br>（3）$y = \pm\sqrt{x}$<br>**解**：<br>（1）是函数，因为对于每个$x \in R$，都有唯一的$y = x^2 \geq 0$与之对应<br>（2）不是函数，因为对于$x = 0$，有$y = \pm 1$两个值<br>（3）不是函数，因为对于$x > 0$，有$y = \pm\sqrt{x}$两个值<br>**例2** 求函数$f(x) = \frac{1}{\sqrt{x-1}}$的定义域。<br>**解**：要使函数有意义，必须：<br>1. $x - 1 > 0$，即$x > 1$<br>2. $\sqrt{x-1} \neq 0$，即$x \neq 1$<br>综合得定义域为$D = (1, +\infty)$<br>【学生】完成函数概念练习 | 深入学习函数的基本概念 |
| 函数的基本性质（30min） | 【教师】详细讲解函数的基本性质<br>**1. 有界性**<br>如果存在常数$M$，使得$|f(x)| \leq M$对所有$x \in D$成立，则称$f(x)$有界。<br>**2. 单调性**<br>如果对于$x_1, x_2 \in D$，当$x_1 < x_2$时，有$f(x_1) < f(x_2)$，则称$f(x)$单调递增；<br>如果对于$x_1, x_2 \in D$，当$x_1 < x_2$时，有$f(x_1) > f(x_2)$，则称$f(x)$单调递减。<br>**3. 奇偶性**<br>如果$f(-x) = f(x)$对所有$x \in D$成立，则称$f(x)$为偶函数；<br>如果$f(-x) = -f(x)$对所有$x \in D$成立，则称$f(x)$为奇函数。<br>**4. 周期性**<br>如果存在常数$T > 0$，使得$f(x + T) = f(x)$对所有$x \in D$成立，则称$f(x)$为周期函数，$T$称为周期。<br>【教师】使用Lab1-2展示函数的性质<br>【学生】观察函数性质的几何表示<br>**例3** 判断函数$f(x) = x^2$的性质。<br>**解**：<br>1. **有界性**：在有限区间上有界，在无穷区间上无界<br>2. **单调性**：在$(-\infty, 0]$上单调递减，在$[0, +\infty)$上单调递增<br>3. **奇偶性**：$f(-x) = (-x)^2 = x^2 = f(x)$，所以是偶函数<br>4. **周期性**：不是周期函数<br>**例4** 判断函数$f(x) = \sin x$的性质。<br>**解**：<br>1. **有界性**：$|\sin x| \leq 1$，所以有界<br>2. **单调性**：在$[-\frac{\pi}{2}, \frac{\pi}{2}]$上单调递增，在$[\frac{\pi}{2}, \frac{3\pi}{2}]$上单调递减<br>3. **奇偶性**：$f(-x) = \sin(-x) = -\sin x = -f(x)$，所以是奇函数<br>4. **周期性**：$f(x + 2\pi) = \sin(x + 2\pi) = \sin x = f(x)$，所以是周期函数，周期为$2\pi$<br>【学生】完成函数性质练习 | 深入掌握函数的基本性质 |
| 基本初等函数（35min） | 【教师】详细讲解基本初等函数<br>**1. 幂函数**<br>$y = x^\alpha$（$\alpha$为常数）<br>- 当$\alpha > 0$时，函数在$(0, +\infty)$上单调递增<br>- 当$\alpha < 0$时，函数在$(0, +\infty)$上单调递减<br>- 当$\alpha = 0$时，$y = 1$（常数函数）<br>**2. 指数函数**<br>$y = a^x$（$a > 0, a \neq 1$）<br>- 定义域：$(-\infty, +\infty)$<br>- 值域：$(0, +\infty)$<br>- 当$a > 1$时，函数单调递增<br>- 当$0 < a < 1$时，函数单调递减<br>**3. 对数函数**<br>$y = \log_a x$（$a > 0, a \neq 1$）<br>- 定义域：$(0, +\infty)$<br>- 值域：$(-\infty, +\infty)$<br>- 当$a > 1$时，函数单调递增<br>- 当$0 < a < 1$时，函数单调递减<br>**4. 三角函数**<br>$y = \sin x, \cos x, \tan x, \cot x, \sec x, \csc x$<br>- $\sin x, \cos x$：定义域$(-\infty, +\infty)$，值域$[-1, 1]$<br>- $\tan x$：定义域$x \neq k\pi + \frac{\pi}{2}$，值域$(-\infty, +\infty)$<br>**5. 反三角函数**<br>$y = \arcsin x, \arccos x, \arctan x, \text{arccot } x$<br>- $\arcsin x$：定义域$[-1, 1]$，值域$[-\frac{\pi}{2}, \frac{\pi}{2}]$<br>- $\arccos x$：定义域$[-1, 1]$，值域$[0, \pi]$<br>- $\arctan x$：定义域$(-\infty, +\infty)$，值域$(-\frac{\pi}{2}, \frac{\pi}{2})$<br>【教师】使用Lab1-3展示基本初等函数<br>【学生】观察基本初等函数的图像<br>**例5** 画出函数$y = 2^x$的图像并分析其性质。<br>**解**：$y = 2^x$是指数函数<br>- 定义域为$(-\infty, +\infty)$<br>- 值域为$(0, +\infty)$<br>- 单调递增<br>- 当$x \to -\infty$时，$y \to 0$；当$x \to +\infty$时，$y \to +\infty$<br>- 图像经过点$(0, 1)$<br>**例6** 求函数$y = \log_2(x-1)$的定义域和值域。<br>**解**：<br>- 定义域：$x - 1 > 0$，即$x > 1$，所以$D = (1, +\infty)$<br>- 值域：由于$\log_2(x-1)$可以取到所有实数值，所以值域为$R = (-\infty, +\infty)$<br>【学生】完成基本初等函数练习 | 深入学习基本初等函数 |
| 复合函数（25min） | 【教师】详细讲解复合函数的概念<br>**定义2** 设$y = f(u)$，$u = g(x)$，则$y = f[g(x)]$称为复合函数。<br>其中$u$称为中间变量。<br>【教师】讲解复合函数的求导<br>如果$y = f(u)$，$u = g(x)$都可导，则<br>$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$<br>这就是链式法则。<br>【教师】讲解复合函数的定义域<br>复合函数$y = f[g(x)]$的定义域是使得$g(x)$的值域包含在$f(u)$的定义域内的所有$x$的集合。<br>【教师】使用Lab1-4展示复合函数<br>【学生】观察复合函数的构成<br>**例7** 求复合函数$y = \sin(x^2)$的导数。<br>**解**：设$u = x^2$，则$y = \sin u$<br>$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = \cos u \cdot 2x = 2x\cos(x^2)$<br>**例8** 求复合函数$y = e^{x^2 + 1}$的导数。<br>**解**：设$u = x^2 + 1$，则$y = e^u$<br>$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = e^u \cdot 2x = 2xe^{x^2 + 1}$<br>**例9** 求复合函数$y = \ln(\sin x)$的定义域。<br>**解**：要使函数有意义，必须$\sin x > 0$<br>所以定义域为$D = \{x | 2k\pi < x < (2k+1)\pi, k \in Z\}$<br>【学生】完成复合函数练习 | 深入学习复合函数的概念 |
| 课堂测验（15min） | 【教师】出几道测试题目<br>1. 判断$y = x^2$是否为函数，并说明理由<br>2. 求函数$f(x) = \frac{1}{\sqrt{x-1}}$的定义域<br>3. 判断函数$f(x) = \sin x$的奇偶性和周期性<br>4. 画出函数$y = 2^x$的图像并分析其性质<br>5. 求复合函数$y = \sin(x^2)$的导数<br>【学生】做测试题目<br>【教师】公布答案并详细讲解 | 通过测试，了解学生对知识点的掌握情况 |
| 课堂小结（10min） | 【教师】总结本节课要点<br>1. 函数的概念和表示方法<br>2. 函数的基本性质（有界性、单调性、奇偶性、周期性）<br>3. 基本初等函数（幂函数、指数函数、对数函数、三角函数、反三角函数）<br>4. 复合函数的概念和求导<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |
