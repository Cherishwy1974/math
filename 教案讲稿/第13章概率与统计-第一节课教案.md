# 第十三章《概率与统计》第一节课教案

## 教学项目

**教学项目** 概率与统计基础

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生具备微积分与线性代数基础，但概率统计经验有限。学生对随机性概念理解不够深入，在条件概率推理、分布参数与实际情境对应、置信区间与假设检验步骤方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生理解概率的基本概念、概率的加法与乘法公式。  
能力目标：培养学生利用概率的基本公式解决实际工程问题的能力。  
素质目标：提高学生的数据思维能力和概率分析意识，在工程场景中合理评估风险与不确定性。

**教学重点** 概率的基本概念、概率的加法与乘法公式。

**教学难点及应对**  
难点：概率概念的理解、事件关系的判断。  
应对策略：通过具体的例题演示，分步骤讲解，辅以Lab13系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第13章概率与统计》、Lab13系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：讲解概率的基本概念、性质及其计算方法。  
问答法：鼓励学生提问，通过问题引导学生思考和深入理解。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab13系列软件演示概率模拟过程。

**教学反思** 需要关注学生对概率概念的理解是否到位，如果发现学生存在困惑，应在下一课时进行针对性的复习和讲解。同时，要注意培养学生通过概率分析解决实际工程问题的能力，强化其应用意识。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |
| --- | --- | --- |
| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 情境导入（10min） | 【教师】讲述液压系统故障案例，引出概率分析需求<br>在工程实践中，我们经常需要分析系统的可靠性。例如，液压系统的密封件在运行过程中可能出现故障，这种故障的发生具有随机性。如何用数学语言来描述和分析这种随机现象？<br>【学生】填写概率树图背景栏，思考如何用概率语言描述"风险"<br>【教师】板书"概率→分布→统计推断"主线<br>【教师】展示工程案例：某工厂生产线故障分析<br>案例背景：某汽车制造厂的生产线由传送带、机械臂、检测设备三个部分组成。根据历史数据，各部分的故障率分别为0.02、0.03、0.01。如何评估整个生产线的可靠性？<br>【学生】讨论分析思路，提出初步解决方案 | 激发学生学习兴趣，建立概率统计与实际工程的联系 |
| 概率基础（35min） | 【教师】讲解概率的基本概念<br>定义1 在相同条件下，可能出现不同结果的现象称为随机现象。<br>例如：抛硬币、掷骰子、产品检验、设备故障等。<br>定义2 随机试验的所有可能结果的集合称为样本空间，记作Ω。<br>例如：掷骰子的样本空间Ω = {1,2,3,4,5,6}<br>定义3 样本空间的子集称为随机事件。<br>例如：A = {2,4,6}（掷出偶数）<br>定义4 必然事件：Ω（一定发生）<br>不可能事件：∅（不会发生）<br>【教师】讲解概率的定义<br>古典概率：P(A) = 事件A包含的基本事件数/样本空间的基本事件总数<br>几何概率：P(A) = 事件A的几何度量/样本空间的几何度量<br>统计概率：P(A) = lim(n→∞) m/n，其中m为事件A发生的次数，n为试验总次数<br>【教师】演示Lab13-1模拟，讲解频率与概率关系<br>频率 = 事件发生次数/试验总次数<br>当试验次数n很大时，频率趋于稳定，这个稳定值称为概率。<br>大数定律：lim(n→∞) P(\|m/n - p\| < ε) = 1<br>【学生】调整试验次数，记录频率稳定趋势，讨论频率趋于稳定的意义<br>【教师】强调经典概型与频率估计的差异<br>例1 掷一枚均匀骰子，求出现偶数的概率。<br>解：样本空间Ω = {1,2,3,4,5,6}，事件A = {2,4,6}<br>P(A) = \|A\|/\|Ω\| = 3/6 = 1/2<br>例2 从52张扑克牌中随机抽取一张，求抽到红桃的概率。<br>解：样本空间包含52个基本事件，事件A包含13个基本事件<br>P(A) = 13/52 = 1/4<br>例3 在区间[0,1]上随机取一点，求该点落在[0.3,0.7]内的概率。<br>解：这是几何概率问题<br>P(A) = [0.7-0.3]/[1-0] = 0.4/1 = 0.4<br>例4 某工厂生产的产品，经检验合格率为0.95。现从产品中随机抽取100件，求其中恰好95件合格的概率。<br>解：设X为合格品数，X ~ B(100, 0.95)<br>P(X=95) = C₁₀₀⁹⁵ × 0.95⁹⁵ × 0.05⁵ ≈ 0.179<br>例5 某设备在一年内发生故障的概率为0.1，求连续两年都不发生故障的概率。<br>解：设A₁ = {第一年无故障}，A₂ = {第二年无故障}<br>P(A₁A₂) = P(A₁)P(A₂) = 0.9 × 0.9 = 0.81<br>例6 某系统由三个部件组成，各部件故障率分别为0.01、0.02、0.03。若系统串联，求系统可靠度。<br>解：P(系统正常) = P(部件1正常)P(部件2正常)P(部件3正常)<br>                = 0.99 × 0.98 × 0.97 = 0.941<br>例7 若系统并联，求系统可靠度。<br>解：P(系统正常) = 1 - P(系统故障) = 1 - P(所有部件都故障)<br>                = 1 - 0.01 × 0.02 × 0.03 = 1 - 0.000006 = 0.999994<br>【学生】完成概率计算练习，讨论不同概率模型的应用场景 | 学习概率的基本概念，通过仿真实验加深理解 |
| 加法与乘法公式（25min） | 【教师】讲解概率的基本公式<br>定理1（概率的加法公式）<br>P(A∪B) = P(A) + P(B) - P(A∩B)<br>当A、B互斥时：P(A∪B) = P(A) + P(B)<br>当A、B对立时：P(A) + P(B) = 1<br>定理2（概率的乘法公式）<br>P(AB) = P(A)P(B\|A) = P(B)P(A\|B)<br>当A、B独立时：P(AB) = P(A)P(B)<br>【教师】讲解事件的关系<br>互斥事件：A∩B = ∅，不能同时发生<br>对立事件：A∪B = Ω 且 A∩B = ∅，必有一个发生<br>独立事件：P(B\|A) = P(B)，A的发生不影响B的概率<br>【教师】使用Lab13-2展示文氏图与模拟结果，解释互斥、独立情况<br>【学生】在学习单中完成公式验证并说明独立/互斥条件<br>例8 某工厂有两个车间，第一车间产品合格率为0.95，第二车间产品合格率为0.90。从第一车间随机抽取一件产品，求它是合格品的概率。<br>解：设A = {从第一车间抽取}，B = {产品合格}<br>P(B\|A) = 0.95<br>例9 从52张扑克牌中不放回地抽取两张，求第一张是红桃且第二张也是红桃的概率。<br>解：设A = {第一张是红桃}，B = {第二张是红桃}<br>P(AB) = P(A)P(B\|A) = (13/52) × (12/51) = 1/17<br>例10 某设备在一年内发生故障的概率为0.1，求两年内都不发生故障的概率。<br>解：设A₁ = {第一年无故障}，A₂ = {第二年无故障}<br>P(A₁A₂) = P(A₁)P(A₂) = 0.9 × 0.9 = 0.81（独立事件）<br>例11 某系统由两个部件组成，部件1的可靠度为0.9，部件2的可靠度为0.8。若两个部件串联，求系统可靠度。<br>解：设A₁ = {部件1正常}，A₂ = {部件2正常}<br>P(A₁A₂) = P(A₁)P(A₂) = 0.9 × 0.8 = 0.72<br>例12 若两个部件并联，求系统可靠度。<br>解：P(A₁∪A₂) = P(A₁) + P(A₂) - P(A₁A₂) = 0.9 + 0.8 - 0.72 = 0.98<br>例13 某工程项目面临三种风险：技术风险、市场风险、资金风险。根据历史数据，各风险发生的概率分别为0.3、0.2、0.1。若技术风险发生，项目损失为100万元；若市场风险发生，项目损失为80万元；若资金风险发生，项目损失为60万元。各风险相互独立。求项目期望损失。<br>解：E(X) = 100×0.3 + 80×0.2 + 60×0.1 = 30 + 16 + 6 = 52万元<br>【学生】分组讨论，完成复杂概率计算题目 | 掌握概率的基本运算规则，培养工程应用意识 |
| 课堂测验（10min） | 【教师】出几道测试题目，测试大家的学习情况<br>1. 掷一枚骰子，求出现偶数的概率<br>2. 从52张扑克牌中随机抽取一张，求抽到红桃或A的概率<br>3. 某设备故障率为0.05，求连续运行10天不故障的概率<br>4. 某系统由两个独立部件组成，部件1可靠度为0.9，部件2可靠度为0.8。求系统串联和并联时的可靠度<br>【学生】做测试题目<br>【教师】公布题目正确答案，并演示解题过程<br>解1：P(偶数) = 3/6 = 1/2<br>解2：P(红桃或A) = P(红桃) + P(A) - P(红桃A) = 13/52 + 4/52 - 1/52 = 16/52 = 4/13<br>解3：P(10天不故障) = 0.95¹⁰ ≈ 0.599<br>解4：串联：0.9×0.8 = 0.72；并联：0.9+0.8-0.72 = 0.98 | 通过测试，了解学生对知识点的掌握情况，加深学生对本节课知识的印象 |
| 课堂小结（8min） | 【教师】总结本节课要点<br>1. 概率的基本概念和定义<br>2. 概率的加法公式和乘法公式<br>3. 事件的关系（互斥、对立、独立）<br>4. 工程应用案例<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |

## 板书设计建议

**左侧**：概率基本概念（随机现象、样本空间、随机事件）
**中部**：概率公式（加法公式、乘法公式）
**右侧**：事件关系（互斥、对立、独立）

## 教学提示

- 鼓励学生截图Lab13模拟结果作为报告证据，提升数据说服力
- 引导学生在概率计算中注意事件关系的判断
- 结合液压系统故障案例，让学生体验概率分析在工程决策中的重要作用
- 强调概率分析在工程质量管理、可靠性分析、风险评估中的重要作用
- 通过实际工程案例，培养学生运用概率方法解决实际问题的能力
