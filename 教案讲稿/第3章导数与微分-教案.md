# 第三章《导数与微分》教案

## 教学项目

**教学项目** 导数与微分

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已掌握极限的基本概念和计算方法，但对导数和微分的理解还不够深入。学生在导数的概念、导数的计算、微分的概念和应用方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生深入理解导数和微分的概念、掌握导数的计算方法和微分的应用。  
能力目标：培养学生利用导数和微分解决实际问题的能力、提高数学分析能力。  
素质目标：提高学生的数学抽象思维和计算能力，培养严谨的数学态度。

**教学重点** 导数的概念、导数的计算、微分的概念和应用、导数的几何意义。

**教学难点及应对**  
难点：导数的概念理解、复合函数求导、微分的概念和应用。  
应对策略：通过具体的工程案例演示，分步骤详细讲解，辅以Lab3系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第3章导数与微分》、Lab3系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：详细讲解导数和微分的概念和计算方法。  
问答法：鼓励学生提问，通过问题引导学生深入思考。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab3系列软件演示导数和微分过程。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |
| --- | --- | --- |
| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 情境导入（15min） | 【教师】讲述导数在工程中的应用案例<br>在工程实践中，我们经常需要分析变化率，比如速度、加速度、增长率、边际成本等。如何用数学方法来解决这些问题？<br>【学生】思考并讨论导数的实际意义<br>【教师】展示工程案例：<br>1. 瞬时速度：v = lim(Δt→0) Δs/Δt<br>2. 瞬时加速度：a = lim(Δt→0) Δv/Δt<br>3. 边际成本：MC = lim(Δq→0) ΔC/Δq<br>4. 温度变化率：dT/dt = k(T - T₀)<br>【学生】讨论分析思路，提出初步解决方案<br>【教师】板书"导数→变化率→应用"主线 | 激发学生学习兴趣，建立导数与实际工程的联系 |
| 导数的概念（35min） | 【教师】详细讲解导数的定义<br>**定义1** 设函数$y = f(x)$在点$x_0$的某个邻域内有定义，如果极限<br>$\lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$<br>存在，则称此极限为函数$f(x)$在点$x_0$处的导数，记作$f'(x_0)$或$\frac{dy}{dx}\bigg|_{x = x_0}$。<br>【教师】讲解导数的几何意义<br>导数$f'(x_0)$表示曲线$y = f(x)$在点$(x_0, f(x_0))$处的切线斜率。<br>【教师】讲解导数的物理意义<br>导数表示函数在某点的瞬时变化率。<br>【教师】讲解导数的计算方法<br>1. **定义法**：直接使用导数定义<br>2. **公式法**：使用基本导数公式<br>3. **运算法则**：使用导数的运算法则<br>【教师】使用Lab3-1展示导数的几何意义<br>【学生】观察导数的几何表示<br>**例1** 求函数$f(x) = x^2$在点$x = 1$处的导数。<br>**解**：$f'(1) = \lim_{\Delta x \to 0} \frac{f(1 + \Delta x) - f(1)}{\Delta x}$<br>$= \lim_{\Delta x \to 0} \frac{(1 + \Delta x)^2 - 1}{\Delta x}$<br>$= \lim_{\Delta x \to 0} \frac{1 + 2\Delta x + (\Delta x)^2 - 1}{\Delta x}$<br>$= \lim_{\Delta x \to 0} \frac{2\Delta x + (\Delta x)^2}{\Delta x}$<br>$= \lim_{\Delta x \to 0} (2 + \Delta x) = 2$<br>**例2** 求函数$f(x) = \sin x$在点$x = 0$处的导数。<br>**解**：$f'(0) = \lim_{\Delta x \to 0} \frac{f(0 + \Delta x) - f(0)}{\Delta x}$<br>$= \lim_{\Delta x \to 0} \frac{\sin(\Delta x) - \sin 0}{\Delta x}$<br>$= \lim_{\Delta x \to 0} \frac{\sin(\Delta x)}{\Delta x} = 1$<br>【学生】完成导数概念练习 | 深入学习导数的基本概念 |
| 导数的计算（35min） | 【教师】详细讲解基本导数公式<br>1. $(x^n)' = nx^{n-1}$<br>2. $(a^x)' = a^x \ln a$<br>3. $(e^x)' = e^x$<br>4. $(\log_a x)' = \frac{1}{x \ln a}$<br>5. $(\ln x)' = \frac{1}{x}$<br>6. $(\sin x)' = \cos x$<br>7. $(\cos x)' = -\sin x$<br>8. $(\tan x)' = \sec^2 x$<br>9. $(\cot x)' = -\csc^2 x$<br>10. $(\sec x)' = \sec x \tan x$<br>11. $(\csc x)' = -\csc x \cot x$<br>12. $(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$<br>13. $(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$<br>14. $(\arctan x)' = \frac{1}{1+x^2}$<br>【教师】详细讲解导数的运算法则<br>**定理1** 如果$u = u(x)$，$v = v(x)$都可导，则：<br>1. $(u \pm v)' = u' \pm v'$<br>2. $(uv)' = u'v + uv'$<br>3. $(\frac{u}{v})' = \frac{u'v - uv'}{v^2}$（$v \neq 0$）<br>4. $(f(g(x)))' = f'(g(x)) \cdot g'(x)$（链式法则）<br>【教师】使用Lab3-2展示导数的计算过程<br>【学生】观察导数的计算步骤<br>**例3** 求函数$f(x) = x^3 + 2x^2 + 3x + 1$的导数。<br>**解**：$f'(x) = (x^3)' + (2x^2)' + (3x)' + (1)'$<br>$= 3x^2 + 4x + 3 + 0 = 3x^2 + 4x + 3$<br>**例4** 求函数$f(x) = x^2 \sin x$的导数。<br>**解**：$f'(x) = (x^2)' \sin x + x^2 (\sin x)'$<br>$= 2x \sin x + x^2 \cos x$<br>**例5** 求函数$f(x) = \frac{x^2 + 1}{x - 1}$的导数。<br>**解**：$f'(x) = \frac{(x^2 + 1)'(x - 1) - (x^2 + 1)(x - 1)'}{(x - 1)^2}$<br>$= \frac{2x(x - 1) - (x^2 + 1) \cdot 1}{(x - 1)^2}$<br>$= \frac{2x^2 - 2x - x^2 - 1}{(x - 1)^2} = \frac{x^2 - 2x - 1}{(x - 1)^2}$<br>【学生】完成导数计算练习 | 深入掌握导数的计算方法 |
| 复合函数求导（25min） | 【教师】详细讲解复合函数求导（链式法则）<br>**定理2（链式法则）** 如果$y = f(u)$，$u = g(x)$都可导，则<br>$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$<br>【教师】讲解复合函数求导的步骤<br>1. 识别复合函数的结构<br>2. 设中间变量<br>3. 分别求导<br>4. 相乘得到结果<br>【教师】使用Lab3-3展示复合函数求导<br>【学生】观察复合函数求导的过程<br>**例6** 求复合函数$y = \sin(x^2)$的导数。<br>**解**：设$u = x^2$，则$y = \sin u$<br>$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = \cos u \cdot 2x = 2x\cos(x^2)$<br>**例7** 求复合函数$y = e^{x^2 + 1}$的导数。<br>**解**：设$u = x^2 + 1$，则$y = e^u$<br>$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = e^u \cdot 2x = 2xe^{x^2 + 1}$<br>**例8** 求复合函数$y = \ln(\sin x)$的导数。<br>**解**：设$u = \sin x$，则$y = \ln u$<br>$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = \frac{1}{u} \cdot \cos x = \frac{\cos x}{\sin x} = \cot x$<br>【学生】完成复合函数求导练习 | 掌握复合函数求导方法 |
| 微分的概念（20min） | 【教师】详细讲解微分的定义<br>**定义2** 设函数$y = f(x)$在点$x_0$处可导，则$f'(x_0)\Delta x$称为函数$f(x)$在点$x_0$处的微分，记作$dy = f'(x_0)dx$。<br>【教师】讲解微分的几何意义<br>微分$dy$表示函数$y = f(x)$在点$(x_0, f(x_0))$处的切线在$x$方向上的增量。<br>【教师】讲解微分的性质<br>1. $d(c) = 0$（常数函数的微分为零）<br>2. $d(cu) = c \cdot du$（常数倍的微分）<br>3. $d(u \pm v) = du \pm dv$（和差的微分）<br>4. $d(uv) = u \cdot dv + v \cdot du$（积的微分）<br>5. $d(\frac{u}{v}) = \frac{v \cdot du - u \cdot dv}{v^2}$（商的微分）<br>【教师】使用Lab3-4展示微分的几何意义<br>【学生】观察微分的几何表示<br>**例9** 求函数$y = x^2$在点$x = 1$处的微分。<br>**解**：$y' = 2x$，所以$dy = 2x \cdot dx$<br>在点$x = 1$处，$dy = 2 \cdot 1 \cdot dx = 2dx$<br>**例10** 求函数$y = e^x$在点$x = 0$处的微分。<br>**解**：$y' = e^x$，所以$dy = e^x \cdot dx$<br>在点$x = 0$处，$dy = e^0 \cdot dx = dx$<br>【学生】完成微分概念练习 | 学习微分的基本概念 |
| 微分的应用（15min） | 【教师】详细讲解微分的应用<br>1. **近似计算**：$f(x_0 + \Delta x) \approx f(x_0) + f'(x_0)\Delta x$<br>2. **误差估计**：$\Delta y \approx dy = f'(x)dx$<br>3. **相对误差**：$\frac{\Delta y}{y} \approx \frac{dy}{y} = \frac{f'(x)dx}{f(x)}$<br>【教师】使用Lab3-5展示微分的应用<br>【学生】观察微分的实际应用<br>**例11** 计算$\sqrt{4.01}$的近似值。<br>**解**：设$f(x) = \sqrt{x}$，则$f'(x) = \frac{1}{2\sqrt{x}}$<br>$f(4.01) \approx f(4) + f'(4) \cdot 0.01$<br>$= 2 + \frac{1}{2 \cdot 2} \cdot 0.01 = 2 + 0.0025 = 2.0025$<br>**例12** 计算$\sin 31°$的近似值。<br>**解**：设$f(x) = \sin x$，则$f'(x) = \cos x$<br>$f(31°) \approx f(30°) + f'(30°) \cdot 1°$<br>$= \sin 30° + \cos 30° \cdot \frac{\pi}{180}$<br>$= \frac{1}{2} + \frac{\sqrt{3}}{2} \cdot \frac{\pi}{180} \approx 0.5 + 0.0151 = 0.5151$<br>【学生】完成微分应用练习 | 学习微分的实际应用 |
| 课堂测验（15min） | 【教师】出几道测试题目<br>1. 求函数$f(x) = x^2$在点$x = 1$处的导数<br>2. 求函数$f(x) = x^3 + 2x^2 + 3x + 1$的导数<br>3. 求函数$f(x) = \sin(x^2)$的导数<br>4. 求函数$f(x) = e^{x^2 + 1}$的导数<br>5. 计算$\sqrt{4.01}$的近似值<br>【学生】做测试题目<br>【教师】公布答案并详细讲解 | 通过测试，了解学生对知识点的掌握情况 |
| 课堂小结（10min） | 【教师】总结本节课要点<br>1. 导数的概念和几何意义<br>2. 导数的计算方法<br>3. 复合函数求导（链式法则）<br>4. 微分的概念和应用<br>5. 导数和微分的实际应用<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |

- 引导学生在导数计算中注意链式法则的应用
- 结合工程案例，让学生体验导数在工程中的重要作用
- 强调导数在微积分学中的基础作用
- 通过实际工程案例，培养学生运用导数方法解决实际问题的能力
