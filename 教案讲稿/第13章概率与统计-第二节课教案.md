# 第十三章《概率与统计》第二节课教案

## 教学项目

**教学项目** 条件概率与贝叶斯公式、随机变量与分布

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已掌握概率的基本概念和运算规则，但对条件概率的理解还不够深入，对随机变量的概念和常见分布的特征需要进一步学习。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生理解条件概率与贝叶斯公式、随机变量的概念、常见概率分布的特征。  
能力目标：培养学生利用条件概率和概率分布解决实际工程问题的能力。  
素质目标：提高学生的数据思维能力和统计推断意识，在工程场景中合理评估风险与不确定性。

**教学重点** 条件概率与贝叶斯公式、常见概率分布。

**教学难点及应对**  
难点：条件概率的树状图推理、分布参数与实际情境对应。  
应对策略：通过具体的例题演示，分步骤讲解，辅以Lab13系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第13章概率与统计》、Lab13系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：讲解条件概率与贝叶斯公式、随机变量及其分布。  
问答法：鼓励学生提问，通过问题引导学生思考和深入理解。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab13系列软件演示概率模拟和分布特征。

**教学反思** 需要关注学生对条件概率推理和分布参数的理解是否到位，如果发现学生存在困惑，应在下一课时进行针对性的复习和讲解。同时，要注意培养学生通过概率分布解决实际工程问题的能力，强化其应用意识。

| 条件概率与贝叶斯（35min） | 【教师】讲解条件概率<br>定义5 在事件B发生的条件下，事件A发生的概率称为条件概率，记作P(A\|B)。<br>P(A\|B) = P(AB)/P(B)，其中P(B) > 0<br>【教师】讲解全概率公式<br>设B₁, B₂, ..., Bₙ是样本空间Ω的一个划分，且P(Bᵢ) > 0，则<br>P(A) = Σᵢ₌₁ⁿ P(A\|Bᵢ)P(Bᵢ)<br>【教师】讲解贝叶斯公式<br>P(Bᵢ\|A) = P(A\|Bᵢ)P(Bᵢ)/P(A) = P(A\|Bᵢ)P(Bᵢ)/Σⱼ₌₁ⁿ P(A\|Bⱼ)P(Bⱼ)<br>【教师】操作Lab13-3构建"工厂—合格/不合格"树状图，演示贝叶斯推理流程<br>【学生】填写概率树图，计算后验概率并解释实际意义<br>例14 某工厂有三个车间生产同一种产品，第一车间产量占总产量的50%，第二车间占30%，第三车间占20%。各车间的次品率分别为2%、3%、1%。现从总产品中随机抽取一件，求它是次品的概率。<br>解：设A = {抽到次品}，Bᵢ = {抽到第i车间产品}，i = 1,2,3<br>P(B₁) = 0.5，P(B₂) = 0.3，P(B₃) = 0.2<br>P(A\|B₁) = 0.02，P(A\|B₂) = 0.03，P(A\|B₃) = 0.01<br>P(A) = P(A\|B₁)P(B₁) + P(A\|B₂)P(B₂) + P(A\|B₃)P(B₃)<br>     = 0.02×0.5 + 0.03×0.3 + 0.01×0.2 = 0.021<br>例15 在例14中，已知抽到的是次品，求它来自第一车间的概率。<br>解：P(B₁\|A) = P(A\|B₁)P(B₁)/P(A) = 0.02×0.5/0.021 = 10/21<br>例16 某疾病在人群中的发病率为0.1%，某种检测方法的准确率为99%（即患者检测为阳性的概率为99%，非患者检测为阴性的概率为99%）。现某人检测为阳性，求他确实患病的概率。<br>解：设A = {患病}，B = {检测阳性}<br>P(A) = 0.001，P(B\|A) = 0.99，P(B\|Ā) = 0.01<br>P(A\|B) = P(B\|A)P(A)/[P(B\|A)P(A) + P(B\|Ā)P(Ā)]<br>       = 0.99×0.001/[0.99×0.001 + 0.01×0.999] = 0.09<br>例17 某系统由三个子系统组成，各子系统的故障率分别为0.01、0.02、0.03。若系统故障，求是第一个子系统故障的概率。<br>解：设A = {系统故障}，Bᵢ = {第i个子系统故障}<br>P(B₁) = 0.01，P(B₂) = 0.02，P(B₃) = 0.03<br>P(A\|B₁) = 1，P(A\|B₂) = 1，P(A\|B₃) = 1<br>P(A) = 0.01 + 0.02 + 0.03 = 0.06<br>P(B₁\|A) = P(A\|B₁)P(B₁)/P(A) = 1×0.01/0.06 = 1/6<br>例18 某工厂生产的产品有A、B、C三个等级，分别占产量的50%、30%、20%。各等级产品的次品率分别为2%、3%、1%。现从产品中随机抽取一件，求：<br>（1）抽到次品的概率<br>（2）若抽到的是次品，求它来自A等级的概率<br>解：（1）P(次品) = 0.02×0.5 + 0.03×0.3 + 0.01×0.2 = 0.021<br>（2）P(A等级\|次品) = 0.02×0.5/0.021 = 10/21<br>【学生】完成条件概率计算练习，讨论贝叶斯公式在工程中的应用 | 学习条件概率与贝叶斯公式，理解其在故障诊断中的应用 |
| 随机变量与分布（30min） | 【教师】讲解随机变量<br>定义6 将随机试验的结果用数字表示的变量称为随机变量。<br>离散型随机变量：取值为有限个或可列个<br>连续型随机变量：取值为连续区间<br>【教师】讲解离散型分布<br>（1）二项分布：X ~ B(n,p)<br>P(X=k) = C_n^k p^k (1-p)^(n-k)，k = 0,1,2,...,n<br>E(X) = np，D(X) = np(1-p)<br>（2）泊松分布：X ~ P(λ)<br>P(X=k) = λ^k e^(-λ)/k!，k = 0,1,2,...<br>E(X) = λ，D(X) = λ<br>（3）几何分布：X ~ G(p)<br>P(X=k) = (1-p)^(k-1)p，k = 1,2,3,...<br>E(X) = 1/p，D(X) = (1-p)/p²<br>【教师】讲解连续型分布<br>（1）正态分布：X ~ N(μ,σ²)<br>f(x) = (1/√(2π)σ)e^(-(x-μ)²/(2σ²))<br>E(X) = μ，D(X) = σ²<br>（2）均匀分布：X ~ U(a,b)<br>f(x) = 1/(b-a)，a ≤ x ≤ b<br>E(X) = (a+b)/2，D(X) = (b-a)²/12<br>（3）指数分布：X ~ E(λ)<br>f(x) = λe^(-λx)，x ≥ 0<br>E(X) = 1/λ，D(X) = 1/λ²<br>【教师】通过Lab13-4、13-5、13-6展示分布特征，调整参数观察变化<br>例19 某工厂产品合格率为0.8，随机抽取10件产品，求恰好8件合格的概率。<br>解：X ~ B(10, 0.8)<br>P(X=8) = C₁₀⁸ × 0.8⁸ × 0.2² = 45 × 0.8⁸ × 0.04 ≈ 0.302<br>例20 某电话交换台每分钟接到的呼叫次数服从参数为3的泊松分布，求一分钟内接到5次呼叫的概率。<br>解：X ~ P(3)<br>P(X=5) = 3⁵ e^(-3)/5! = 243 × e^(-3)/120 ≈ 0.1008<br>例21 某零件长度服从N(10, 0.1²)，求长度在9.8到10.2之间的概率。<br>解：P(9.8 < X < 10.2) = P(-2 < (X-10)/0.1 < 2) = Φ(2) - Φ(-2) = 2Φ(2) - 1 ≈ 0.9544<br>例22 某设备故障间隔时间服从参数为0.1的指数分布，求故障间隔时间超过10小时的概率。<br>解：X ~ E(0.1)<br>P(X > 10) = e^(-0.1×10) = e^(-1) ≈ 0.368<br>例23 某零件加工时间服从U(5,15)分钟，求加工时间在8到12分钟之间的概率。<br>解：P(8 < X < 12) = (12-8)/(15-5) = 4/10 = 0.4<br>例24 某系统由两个部件组成，部件1的可靠度为0.9，部件2的可靠度为0.8。若两个部件串联，求系统可靠度。<br>解：P(系统正常) = P(部件1正常)P(部件2正常) = 0.9 × 0.8 = 0.72<br>例25 若两个部件并联，求系统可靠度。<br>解：P(系统正常) = 1 - P(系统故障) = 1 - P(所有部件都故障)<br>                = 1 - 0.1 × 0.2 = 0.98<br>【学生】完成分布计算练习，讨论不同分布的应用场景 | 掌握常见概率分布的特征和应用场景 |
| 课堂测验（10min） | 【教师】出几道测试题目<br>1. 某工厂产品合格率为0.9，随机抽取10件，求恰好8件合格的概率<br>2. 某零件长度服从N(10,0.1²)，求长度在9.8到10.2之间的概率<br>3. 某设备故障间隔时间服从E(0.2)，求故障间隔时间超过5小时的概率<br>4. 某系统由两个独立部件组成，部件1可靠度为0.9，部件2可靠度为0.8。求系统串联和并联时的可靠度<br>【学生】做测试题目<br>【教师】公布答案并讲解<br>解1：P(X=8) = C₁₀⁸ × 0.9⁸ × 0.1² ≈ 0.194<br>解2：P(9.8 < X < 10.2) = 2Φ(2) - 1 ≈ 0.9544<br>解3：P(X > 5) = e^(-0.2×5) = e^(-1) ≈ 0.368<br>解4：串联：0.9×0.8 = 0.72；并联：0.9+0.8-0.72 = 0.98 | 通过测试，了解学生对知识点的掌握情况 |
| 课堂小结（5min） | 【教师】总结本节课要点<br>1. 条件概率与贝叶斯公式<br>2. 常见概率分布<br>3. 工程应用案例<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |
