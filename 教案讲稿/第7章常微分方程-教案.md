# 第七章《常微分方程》教案

## 教学项目

**教学项目** 常微分方程

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已掌握一元函数微分学的基本概念和计算方法，但对常微分方程的理解还不够深入。学生在微分方程的概念、一阶微分方程、二阶微分方程、数值解法方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生理解常微分方程的概念、掌握一阶和二阶微分方程的求解方法。  
能力目标：培养学生利用常微分方程解决实际问题的能力。  
素质目标：提高学生的数学建模思维和工程应用能力，在工程场景中合理运用常微分方程方法。

**教学重点** 常微分方程的概念、一阶微分方程的求解、二阶微分方程的求解。

**教学难点及应对**  
难点：微分方程的建立、求解方法的理解。  
应对策略：通过具体的工程案例演示，分步骤讲解，辅以Lab7系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第7章常微分方程》、Lab7系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：讲解常微分方程的概念和求解方法。  
问答法：鼓励学生提问，通过问题引导学生思考和深入理解。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab7系列软件演示常微分方程求解过程。

**教学反思** 需要关注学生对常微分方程概念的理解是否到位，如果发现学生存在困惑，应进行针对性的复习和讲解。同时，要注意培养学生通过常微分方程方法解决实际问题的能力，强化其应用意识。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |
| --- | --- | --- |
| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 情境导入（10min） | 【教师】讲述常微分方程在工程中的应用案例<br>在工程实践中，我们经常需要描述和预测各种变化过程，比如人口增长、放射性衰变、电路分析、机械振动等。如何用数学方法来描述这些变化规律？<br>【学生】思考并讨论常微分方程的实际应用<br>【教师】展示工程案例：人口增长模型<br>案例背景：某城市人口增长率为2%，初始人口为100万，如何预测未来人口数量？<br>【学生】讨论分析思路，提出初步解决方案<br>【教师】板书"微分方程→建立模型→求解→应用"主线 | 激发学生学习兴趣，建立常微分方程与实际工程的联系 |
| 常微分方程的概念（20min） | 【教师】讲解常微分方程的定义<br>定义1 含有未知函数及其导数的方程称为微分方程。如果未知函数是一元函数，则称为常微分方程。<br>【教师】讲解微分方程的分类<br>1. 按阶数分类：一阶、二阶、高阶微分方程<br>2. 按线性性分类：线性、非线性微分方程<br>3. 按齐次性分类：齐次、非齐次微分方程<br>【教师】讲解微分方程的解<br>1. 通解：含有任意常数的解<br>2. 特解：满足特定初始条件的解<br>3. 初值问题：给定初始条件的微分方程<br>【教师】使用Lab7-1展示微分方程的概念<br>【学生】观察微分方程的几何表示<br>例1 判断下列方程是否为微分方程：<br>（1）$y' + 2y = 0$<br>（2）$y'' + 3y' + 2y = e^x$<br>（3）$x^2 + y^2 = 1$<br>解：（1）是，一阶线性齐次微分方程<br>（2）是，二阶线性非齐次微分方程<br>（3）不是，这是代数方程<br>例2 验证$y = Ce^{-2x}$是微分方程$y' + 2y = 0$的通解。<br>解：$y' = -2Ce^{-2x}$<br>代入方程：$-2Ce^{-2x} + 2 \cdot Ce^{-2x} = 0$<br>左边 = $-2Ce^{-2x} + 2Ce^{-2x} = 0$ = 右边<br>所以$y = Ce^{-2x}$是方程的通解<br>【学生】完成微分方程概念练习 | 学习常微分方程的基本概念 |
| 一阶微分方程（25min） | 【教师】讲解可分离变量方程<br>定义2 形如$\frac{dy}{dx} = f(x)g(y)$的微分方程称为可分离变量方程。<br>求解步骤：<br>1. 分离变量：$\frac{dy}{g(y)} = f(x)dx$<br>2. 两边积分：$\int \frac{dy}{g(y)} = \int f(x)dx$<br>3. 求出通解<br>【教师】讲解一阶线性微分方程<br>定义3 形如$y' + P(x)y = Q(x)$的微分方程称为一阶线性微分方程。<br>求解步骤：<br>1. 求积分因子：$\mu(x) = e^{\int P(x)dx}$<br>2. 方程两边乘以积分因子<br>3. 左边写成$\frac{d}{dx}[\mu(x)y]$的形式<br>4. 两边积分求解<br>【教师】使用Lab7-2展示一阶微分方程的求解过程<br>【学生】观察一阶微分方程的求解步骤<br>例3 求解微分方程$\frac{dy}{dx} = \frac{y}{x}$。<br>解：这是可分离变量方程<br>分离变量：$\frac{dy}{y} = \frac{dx}{x}$<br>两边积分：$\int \frac{dy}{y} = \int \frac{dx}{x}$<br>$\ln|y| = \ln|x| + C_1$<br>$|y| = e^{\ln|x| + C_1} = e^{C_1}|x|$<br>所以$y = Cx$（$C = \pm e^{C_1}$）<br>例4 求解微分方程$y' + 2y = e^x$。<br>解：这是一阶线性微分方程<br>$P(x) = 2$，$Q(x) = e^x$<br>积分因子：$\mu(x) = e^{\int 2dx} = e^{2x}$<br>方程两边乘以$e^{2x}$：<br>$e^{2x}y' + 2e^{2x}y = e^{3x}$<br>左边：$\frac{d}{dx}[e^{2x}y] = e^{3x}$<br>两边积分：$e^{2x}y = \int e^{3x}dx = \frac{1}{3}e^{3x} + C$<br>所以$y = \frac{1}{3}e^x + Ce^{-2x}$<br>【学生】完成一阶微分方程练习 | 掌握一阶微分方程的求解方法 |
| 二阶微分方程（25min） | 【教师】讲解二阶线性微分方程<br>定义4 形如$y'' + P(x)y' + Q(x)y = R(x)$的微分方程称为二阶线性微分方程。<br>【教师】讲解二阶常系数线性微分方程<br>当$P(x) = p$，$Q(x) = q$为常数时，方程为<br>$y'' + py' + qy = R(x)$<br>【教师】讲解齐次方程的通解<br>对于齐次方程$y'' + py' + qy = 0$：<br>1. 写出特征方程：$r^2 + pr + q = 0$<br>2. 求特征根：$r_1, r_2$<br>3. 根据特征根的情况写出通解：<br>   - 两个不等实根：$y = C_1e^{r_1x} + C_2e^{r_2x}$<br>   - 两个相等实根：$y = (C_1 + C_2x)e^{rx}$<br>   - 共轭复根：$y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$<br>【教师】使用Lab7-3展示二阶微分方程的求解过程<br>【学生】观察二阶微分方程的求解步骤<br>例5 求解微分方程$y'' - 3y' + 2y = 0$。<br>解：这是二阶常系数线性齐次方程<br>特征方程：$r^2 - 3r + 2 = 0$<br>$(r - 1)(r - 2) = 0$<br>特征根：$r_1 = 1$，$r_2 = 2$<br>通解：$y = C_1e^x + C_2e^{2x}$<br>例6 求解微分方程$y'' + 4y' + 4y = 0$。<br>解：特征方程：$r^2 + 4r + 4 = 0$<br>$(r + 2)^2 = 0$<br>特征根：$r_1 = r_2 = -2$（重根）<br>通解：$y = (C_1 + C_2x)e^{-2x}$<br>例7 求解微分方程$y'' + 2y' + 5y = 0$。<br>解：特征方程：$r^2 + 2r + 5 = 0$<br>$r = \frac{-2 \pm \sqrt{4 - 20}}{2} = \frac{-2 \pm 4i}{2} = -1 \pm 2i$<br>特征根：$r_1 = -1 + 2i$，$r_2 = -1 - 2i$（共轭复根）<br>通解：$y = e^{-x}(C_1\cos 2x + C_2\sin 2x)$<br>【学生】完成二阶微分方程练习 | 掌握二阶微分方程的求解方法 |
| 微分方程的应用（15min） | 【教师】讲解微分方程的应用<br>1. 人口增长模型：$\frac{dP}{dt} = kP$<br>2. 放射性衰变：$\frac{dN}{dt} = -\lambda N$<br>3. 弹簧振动：$m\frac{d^2x}{dt^2} + kx = 0$<br>4. 电路分析：$L\frac{d^2q}{dt^2} + R\frac{dq}{dt} + \frac{q}{C} = E(t)$<br>【教师】使用Lab7-4展示微分方程的应用<br>【学生】观察微分方程的实际应用<br>例8 某城市人口增长率为2%，初始人口为100万，求人口增长模型。<br>解：设人口为$P(t)$，增长率为$k = 0.02$<br>微分方程：$\frac{dP}{dt} = 0.02P$<br>初始条件：$P(0) = 100$<br>求解：$\frac{dP}{P} = 0.02dt$<br>$\int \frac{dP}{P} = \int 0.02dt$<br>$\ln P = 0.02t + C$<br>$P = Ce^{0.02t}$<br>由$P(0) = 100$得$C = 100$<br>所以$P(t) = 100e^{0.02t}$<br>【学生】完成微分方程应用练习 | 学习微分方程的实际应用 |
| 课堂测验（10min） | 【教师】出几道测试题目<br>1. 求解微分方程$\frac{dy}{dx} = \frac{y}{x}$<br>2. 求解微分方程$y' + 2y = e^x$<br>3. 求解微分方程$y'' - 3y' + 2y = 0$<br>4. 建立人口增长模型$\frac{dP}{dt} = kP$<br>【学生】做测试题目<br>【教师】公布答案并讲解<br>解1：$y = Cx$<br>解2：$y = \frac{1}{3}e^x + Ce^{-2x}$<br>解3：$y = C_1e^x + C_2e^{2x}$<br>解4：$P(t) = P_0e^{kt}$ | 通过测试，了解学生对知识点的掌握情况 |
| 课堂小结（8min） | 【教师】总结本节课要点<br>1. 常微分方程的概念和分类<br>2. 一阶微分方程的求解方法<br>3. 二阶微分方程的求解方法<br>4. 微分方程的实际应用<br>【学生】回顾知识点，提出疑问<br>【教师】解答学生疑问，布置课后作业 | 巩固本节课所学知识 |
