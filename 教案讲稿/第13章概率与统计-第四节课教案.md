# 第十三章《概率与统计》第四节课教案

## 教学项目

**教学项目** 综合应用与总结

**授课地点** 多媒体教室  
**授课形式** 线下教学

**学情分析** 学生已掌握概率统计的基本概念、常见分布、数字特征和统计推断方法，需要通过综合应用来巩固所学知识，提高解决实际工程问题的能力。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

**教学目标**  
知识目标：使学生综合运用概率统计知识解决复杂工程问题。  
能力目标：培养学生分析问题、建立模型、求解问题的综合能力。  
素质目标：提高学生的数据思维能力和统计推断意识，在工程场景中合理评估风险与不确定性。

**教学重点** 综合应用概率统计方法解决工程问题。

**教学难点及应对**  
难点：复杂问题的建模和求解。  
应对策略：通过具体的工程案例演示，分步骤讲解，辅以Lab13系列仿真实验和小组讨论，加深学生理解。

**教学资源**  
教材：《高等数学》  
媒体资源：课件《第13章概率与统计》、Lab13系列仿真实验  
环境设备：多媒体教室配备投影仪和电脑

**教学方法**  
讲授法：讲解综合应用案例。  
问答法：鼓励学生提问，通过问题引导学生思考和深入理解。  
分组练习法：学生分组完成练习题，互相讨论和解答。  
演示法：通过Lab13系列软件演示综合应用过程。

**教学反思** 需要关注学生对综合应用的理解是否到位，如果发现学生存在困惑，应进行针对性的指导。同时，要注意培养学生通过概率统计解决实际工程问题的能力，强化其应用意识。

## 教学过程

| 时间 | 主要教学内容及步骤 | 设计意图 |
| --- | --- | --- |
| 考勤（2min） | 【教师】清点上课人数，记录好考勤<br>【学生】班干部报请假人员及原因 | 培养学生的组织纪律性，掌握学生的出勤情况 |
| 知识体系回顾（15min） | 【教师】系统回顾概率统计知识体系<br>第一部分：概率论基础<br>- 随机现象与随机试验<br>- 概率的定义与性质<br>- 条件概率与独立性<br>第二部分：随机变量及其分布<br>- 随机变量的概念<br>- 离散型分布（二项、泊松、几何）<br>- 连续型分布（正态、均匀、指数）<br>第三部分：数字特征<br>- 数学期望<br>- 方差与标准差<br>第四部分：统计推断<br>- 参数估计<br>- 假设检验<br>【学生】跟随教师回顾，查漏补缺<br>【教师】解答学生在知识体系理解上的疑问 | 系统梳理知识体系，建立完整框架 |
| 工程案例深入分析（30min） | 【教师】深入分析液压系统可靠性案例<br>案例背景：某工程机械的液压系统由泵、阀、缸三个主要部件组成，各部件的故障率分别为0.01、0.02、0.03。系统采用串联结构，任一部件故障都会导致系统失效。<br>问题1：系统在一年内的故障概率是多少？<br>解：P(系统故障) = 1 - P(系统正常) = 1 - P(泵正常)P(阀正常)P(缸正常)<br>                = 1 - 0.99×0.98×0.97 = 1 - 0.941 = 0.059<br>问题2：如果系统故障，最可能是哪个部件故障？<br>解：设A₁、A₂、A₃分别表示泵、阀、缸故障，B表示系统故障<br>P(A₁\|B) = P(B\|A₁)P(A₁)/P(B) = 1×0.01/0.059 = 0.169<br>P(A₂\|B) = P(B\|A₂)P(A₂)/P(B) = 1×0.02/0.059 = 0.339<br>P(A₃\|B) = P(B\|A₃)P(A₃)/P(B) = 1×0.03/0.059 = 0.508<br>最可能是缸故障<br>问题3：如何设计冗余结构提高系统可靠性？<br>解：采用并联冗余，设需要n个并联子系统<br>单个子系统可靠度 = 0.941<br>n个并联子系统可靠度 = 1 - (1-0.941)ⁿ = 1 - 0.059ⁿ<br>要求：1 - 0.059ⁿ ≥ 0.99<br>即：0.059ⁿ ≤ 0.01<br>n ≥ ln(0.01)/ln(0.059) ≈ 2.3<br>所以需要3个并联的子系统<br>【学生】参与讨论，提出改进方案<br>【教师】引导学生思考其他工程应用场景 | 深入理解概率统计在工程中的应用 |
| 综合题目训练（25min） | 【教师】布置综合训练题目<br>题目：某工程项目面临三种风险：技术风险、市场风险、资金风险。根据历史数据，各风险发生的概率分别为0.3、0.2、0.1。若技术风险发生，项目损失为100万元；若市场风险发生，项目损失为80万元；若资金风险发生，项目损失为60万元。各风险相互独立。求：<br>（1）项目期望损失<br>（2）项目损失超过50万元的概率<br>（3）若购买保险，保险费为期望损失的1.2倍，是否值得购买？<br>（4）如何设计风险控制策略？<br>【学生】独立完成题目<br>【教师】巡视指导，解答疑问<br>解：（1）E(X) = 100×0.3 + 80×0.2 + 60×0.1 = 30 + 16 + 6 = 52万元<br>（2）P(X > 50) = P(技术风险发生) + P(市场风险发生) + P(资金风险发生) = 0.3 + 0.2 + 0.1 = 0.6<br>（3）保险费 = 52 × 1.2 = 62.4万元，期望损失 = 52万元<br>保险费 > 期望损失，从纯数学角度不值得购买，但需考虑风险承受能力<br>（4）风险控制策略：<br>- 技术风险：加强技术研发，提高技术成熟度<br>- 市场风险：进行市场调研，制定灵活的市场策略<br>- 资金风险：建立资金储备，多元化融资渠道<br>【学生】讨论风险控制策略，提出创新方案 | 综合运用所学知识解决实际问题 |
| 课堂测验（15min） | 【教师】出综合测试题目<br>1. 某工厂生产的产品有A、B、C三个等级，分别占产量的50%、30%、20%。各等级产品的次品率分别为2%、3%、1%。现从产品中随机抽取一件，求：<br>（1）抽到次品的概率<br>（2）若抽到的是次品，求它来自A等级的概率<br>2. 某电话交换台每分钟接到的呼叫次数X服从参数为λ=3的泊松分布。求：<br>（1）一分钟内接到5次呼叫的概率<br>（2）一分钟内接到不超过3次呼叫的概率<br>3. 某工厂生产的零件长度要求为10mm，现从产品中随机抽取16件，测得平均长度为10.2mm，标准差为0.3mm。在显著性水平α=0.05下，检验零件平均长度是否为10mm。<br>4. 某设备故障间隔时间服从指数分布，现观测到10次故障时间（小时）：1.2, 2.1, 0.8, 1.5, 2.3, 1.8, 1.1, 2.0, 1.6, 1.9。求平均故障时间的95%置信区间。<br>【学生】完成测试题目<br>【教师】公布答案并详细讲解<br>解1：（1）P(B) = 0.02×0.5 + 0.03×0.3 + 0.01×0.2 = 0.021<br>（2）P(A₁\|B) = 0.02×0.5/0.021 = 10/21<br>解2：（1）P(X=5) = 3⁵e^(-3)/5! ≈ 0.1008<br>（2）P(X≤3) = 13e^(-3) ≈ 0.647<br>解3：t = 2.67 > 2.131，拒绝H₀，认为零件平均长度不等于10mm<br>解4：置信区间：[1.31, 1.95] | 检验学习效果，巩固重点知识 |
| 课程总结（3min） | 【教师】总结本课程要点<br>1. 概率统计的基本概念和运算规则<br>2. 常见概率分布的特征和应用<br>3. 统计推断的基本方法<br>4. 概率统计在工程中的应用<br>【教师】布置课后作业<br>1. 完成综合练习册第13章所有题目<br>2. 设计一个工程可靠性分析案例，运用概率统计方法进行分析<br>3. 撰写学习心得，总结概率统计在工程中的应用<br>【学生】记录作业要求，提出疑问<br>【教师】解答学生疑问，鼓励课后深入学习 | 总结课程内容，布置后续学习任务 |
